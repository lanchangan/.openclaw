## 1. 调制概述

### 1.1 什么是调制

调制是将基带信号（信息）加载到载波信号上的过程：

```
基带信号 × 载波信号 = 已调信号

基带信号: 低频，携带信息
载波信号: 高频，易于传输
已调信号: 适合信道传输的高频信号
```

### 1.2 调制的基本参数

载波信号可表示为：
```
s(t) = A·cos(2πft + φ)

A: 幅度 (Amplitude)
f: 频率 (Frequency)  
φ: 相位 (Phase)
```

调制就是改变这三个参数之一或多个来携带信息。

### 1.3 调制分类

```
调制技术
├── 模拟调制
│   ├── AM (幅度调制)
│   ├── FM (频率调制)
│   └── PM (相位调制)
├── 数字调制
│   ├── ASK (幅度键控)
│   ├── FSK (频率键控)
│   ├── PSK (相位键控)
│   └── QAM (正交幅度调制)
└── 扩频调制
    ├── DSSS (直接序列扩频)
    ├── FHSS (跳频扩频)
    └── OFDM (正交频分复用)
```

## 2. 模拟调制

### 2.1 幅度调制 (AM)

```
AM 信号: s(t) = [1 + m(t)]·A·cos(2πfct)

m(t): 调制信号
A: 载波幅度
fc: 载波频率

示例: 广播电台
- 中波广播: 535-1605 kHz
- 调幅广播带宽: ~10 kHz
```

**AM 特点**：
- 实现简单
- 抗噪声能力差
- 功率效率低
- 频谱效率低

### 2.2 频率调制 (FM)

```
FM 信号: s(t) = A·cos[2πfct + 2πkf∫m(τ)dτ]

kf: 频率偏移常数
频偏: Δf = kf·max|m(t)|

示例: FM 广播
- 调频广播: 88-108 MHz
- 频偏: ±75 kHz
- 信道带宽: 200 kHz
```

**FM 特点**：
- 抗噪声能力强
- 功率效率高
- 频谱效率低
- 捕获效应

### 2.3 相位调制 (PM)

```
PM 信号: s(t) = A·cos[2πfct + kp·m(t)]

kp: 相位偏移常数
相位偏移: Δφ = kp·max|m(t)|
```

## 3. 数字调制

### 3.1 数字调制原理

数字调制用离散的符号表示数字比特：

```
比特流: 1 0 1 1 0 0 1 0 ...

调制: 将比特映射到信号参数
- 每 symbol 携带 k bits
- 符号速率 = 比特率 / k

示例:
BPSK: 1 bit/symbol
QPSK: 2 bits/symbol  
16-QAM: 4 bits/symbol
64-QAM: 6 bits/symbol
```

### 3.2 星座图

星座图是数字调制信号在 I-Q 平面上的表示：

```
          Q
          ↑
          │    ★  ★
          │  ★      ★
    ──────┼──────────→ I
          │  ★      ★
          │    ★  ★
          ↓

每个点代表一个符号
- 横轴 (I): 同相分量
- 纵轴 (Q): 正交分量
```

## 4. 幅度调制

### 4.1 ASK (Amplitude Shift Keying)

```
OOK (On-Off Keying) - 2-ASK:
- 比特 "1": 载波幅度 A
- 比特 "0": 载波幅度 0

信号表示:
s(t) = {
    A·cos(2πfct),  bit = 1
    0,             bit = 0
}

星座图 (只有 I 分量):
    I
    ↑
  1 │ ★
    │
  0 ├───────
    │ ★
    ↓
```

**ASK 特点**：
- 实现简单
- 抗噪声能力差
- 用于光通信、RFID

### 4.2 M-ASK

```
4-ASK (每个符号 2 bits):
幅度等级: A·{±3, ±1}

比特映射:
00 → -3A
01 → -A  
11 → +A
10 → +3A

每符号携带: log2(M) = 2 bits
```

## 5. 频率调制

### 5.1 FSK (Frequency Shift Keying)

```
BFSK (Binary FSK):
- 比特 "1": 频率 f1
- 比特 "0": 频率 f2

信号表示:
s(t) = {
    A·cos(2πf1·t),  bit = 1
    A·cos(2πf2·t),  bit = 0
}

频差: Δf = |f1 - f2|
调制指数: h = Δf / Rb (Rb = 比特率)
```

**FSK 特点**：
- 抗噪声能力较强
- 恒包络，适合非线性放大
- 频谱效率低

### 5.2 M-FSK

```
4-FSK (每个符号 2 bits):
频率: f0, f1, f2, f3

比特映射:
00 → f0
01 → f1
10 → f2
11 → f3

频谱带宽: M·Δf (约)
```

### 5.3 GFSK (Gaussian FSK)

```
GFSK: 在 FSK 前加高斯滤波器

作用:
- 平滑频率跳变
- 减少频谱旁瓣
- 提高频谱效率

应用: Bluetooth, DECT

高斯滤波器:
- BT 乘积: 带宽 × 符号周期
- 典型值: BT = 0.5 (GSM), BT = 0.3 (Bluetooth)
```

## 6. 相位调制

### 6.1 BPSK (Binary PSK)

```
BPSK (每个符号 1 bit):
- 比特 "1": 相位 0°
- 比特 "0": 相位 180°

信号表示:
s(t) = A·cos(2πfct + φ)
φ = {0, π}

星座图:
        Q
        ↑
        │
  π ────┼──── 0    (φ)
   ★    │    ★
        │
        ↓
       -1    1     (I)

符号映射:
- "1" → I = +1, Q = 0
- "0" → I = -1, Q = 0

每符号携带: 1 bit
```

**BPSK 特点**：
- 抗噪声能力最强
- 频谱效率低
- 用于深空通信、低 SNR 场景

### 6.2 QPSK (Quadrature PSK)

```
QPSK (每个符号 2 bits):
相位: {45°, 135°, 225°, 315°} 或 {0°, 90°, 180°, 270°}

星座图:
        Q
        ↑
        │  ★ 10
    01 ★┼──────★ 11
        │
    00 ★┼──────★ 00
        │  ★ 01
        ↓

格雷码映射 (相邻只差 1 bit):
00 → φ = 135°  (I=-1, Q=+1)
01 → φ = 45°   (I=+1, Q=+1)
11 → φ = -45°  (I=+1, Q=-1)
10 → φ = -135° (I=-1, Q=-1)

每符号携带: 2 bits
```

**QPSK 特点**：
- 频谱效率比 BPSK 高一倍
- 抗噪声能力较强
- 广泛应用于卫星通信、WiFi

### 6.3 8-PSK

```
8-PSK (每个符号 3 bits):
相位间隔: 45°

星座图:
        Q
        ↑
     ★  │  ★
   ★────┼────★
        │
   ★────┼────★
     ★  │  ★
        ↓

每符号携带: 3 bits
应用: EDGE (GSM 演进)
```

### 6.4 DQPSK (Differential QPSK)

```
DQPSK: 用相位差表示信息

编码规则:
- 相对前一符号的相位变化
- 不需要绝对相位参考

相位差映射:
00 → Δφ = 0°
01 → Δφ = 90°
11 → Δφ = 180°
10 → Δφ = 270°

优点: 不需要相干解调
缺点: 有误差传播
```

## 7. 正交调制

### 7.1 I-Q 调制原理

```
I-Q 调制器:
           ┌─────┐
    I(t) ──│ ×  ├──┐
           └─────┘  │   ┌─────┐
                    ├───│  +  │── s(t)
           ┌─────┐  │   └─────┘
    Q(t) ──│ ×  ├──┘
           └─────┘
              ↑
           cos(2πfct)  sin(2πfct)

信号表示:
s(t) = I(t)·cos(2πfct) - Q(t)·sin(2πfct)
     = A·cos(2πfct + φ)

其中:
A = √(I² + Q²)
φ = arctan(Q/I)
```

### 7.2 I-Q 解调

```
I-Q 解调器:
           ┌─────┐
    s(t) ──│ ×  │──┬── I(t)
           └─────┘  │
              ↑     └── LPF
           cos(2πfct)

           ┌─────┐
    s(t) ──│ ×  │──┬── Q(t)
           └─────┘  │
              ↑     └── LPF
          -sin(2πfct)
```

## 8. QAM 调制详解

### 8.1 QAM 原理

QAM 同时调制 I 和 Q 两个分量：

```
QAM 信号:
s(t) = I·cos(2πfct) - Q·sin(2πfct)

其中 I 和 Q 各有多个电平:
- 16-QAM: I, Q ∈ {±1, ±3}  (4 电平)
- 64-QAM: I, Q ∈ {±1, ±3, ±5, ±7}  (8 电平)
- 256-QAM: I, Q ∈ {±1, ±3, ..., ±15}  (16 电平)
```

### 8.2 16-QAM

```
16-QAM (每个符号 4 bits):

星座图:
        Q
        ↑
    ★   ★ │ ★   ★     (±3, ±3), (±1, ±3), (±3, ±1), (±1, ±1)
  ★───★───┼───★───★
    ★   ★ │ ★   ★
          │
  ★───★───┼───★───★
    ★   ★ │ ★   ★
        ↓

I: {-3, -1, +1, +3}
Q: {-3, -1, +1, +3}

格雷码映射示例:
0000 → I=-3, Q=-3
0001 → I=-1, Q=-3
0011 → I=+1, Q=-3
0010 → I=+3, Q=-3
...

每符号携带: 4 bits
```

### 8.3 64-QAM

```
64-QAM (每个符号 6 bits):

I, Q: {-7, -5, -3, -1, +1, +3, +5, +7}

星座点数: 8 × 8 = 64
每符号携带: 6 bits

WiFi 5 (802.11ac) 最高使用 256-QAM
WiFi 6 (802.11ax) 最高使用 1024-QAM
WiFi 7 (802.11be) 最高使用 4096-QAM
```

### 8.4 QAM 阶数对比

| 调制 | 星座点 | bits/symbol | I/Q 电平数 | 相对 SNR 要求 |
|------|--------|-------------|-----------|--------------|
| BPSK | 2 | 1 | 1 | 0 dB (基准) |
| QPSK | 4 | 2 | 2 | +3 dB |
| 16-QAM | 16 | 4 | 4 | +10 dB |
| 64-QAM | 64 | 6 | 8 | +16 dB |
| 256-QAM | 256 | 8 | 16 | +22 dB |
| 1024-QAM | 1024 | 10 | 32 | +28 dB |
| 4096-QAM | 4096 | 12 | 64 | +35 dB |

### 8.5 QAM 星座点距离

```
相邻星座点距离:
d = 2A / (√M - 1)

M: QAM 阶数
A: 最大幅度

16-QAM:   d = 2A/3
64-QAM:   d = 2A/7
256-QAM:  d = 2A/15
1024-QAM: d = 2A/31
4096-QAM: d = 2A/63

距离越小，对 SNR 要求越高
```

### 8.6 EVM (Error Vector Magnitude)

```
EVM: 衡量调制质量的指标

定义:
EVM = √(平均误差功率 / 平均参考功率)

计算:
误差向量 = 实际星座点 - 理想星座点
EVM = |误差向量| / |理想向量|

示例要求:
调制      EVM 要求
QPSK     -13 dB
16-QAM   -16 dB
64-QAM   -22 dB
256-QAM  -28 dB
1024-QAM -35 dB
4096-QAM -40 dB
```
