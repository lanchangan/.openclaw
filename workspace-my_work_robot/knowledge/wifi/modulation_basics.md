# 调制基础

> 将信息承载到满足信道要求的高频信号上的过程

---

## 一句话总结

> **调制**是用基带信号控制载波的某个参数（幅度、频率、相位），**解调**是逆过程。

---

## 核心概念

### 关键术语

| 术语 | 说明 |
|------|------|
| **调制信号** | 被调制的信号（基带信号） |
| **载波信号** | 无信息量的高频信号 |
| **已调信号** | 调制后得到的新高频信号（频带信号） |

---

## 模拟调制 vs 数字调制

**区分标准**：取决于被调信号是模拟还是数字信号

| 类型 | 调制信号 | 载波信号 |
|------|----------|----------|
| 模拟调制 | 低频模拟信号 | 高频正弦波 |
| 数字调制 | 数字脉冲序列（01序列） | 高频正弦波 |

---

## 三种基本调制方式

### 模拟调制

| 方式 | 英文 | 原理 |
|------|------|------|
| 幅度调制 | AM | 用模拟信号控制载波**幅度** |
| 频率调制 | FM | 用模拟信号控制载波**频率** |
| 相位调制 | PM | 用模拟信号控制载波**相位** |

### 数字调制（键控信号）

| 方式 | 英文 | 原理 | 示例 |
|------|------|------|------|
| 幅度键控 | ASK | 用不同幅度代表 0 和 1 | 高电平=1, 低电平=0 |
| 频移键控 | FSK | 用不同频率代表 0 和 1 | f1=1, f2=0 |
| 相移键控 | PSK | 用不同相位代表 0 和 1 | 0°=0, 180°=1 |

> 💡 数字调制又称"键控信号"，因为离散状态像开关按键控制"通与断"

## WiFi 中的调制演进

| WiFi 代际 | 协议 | 最高调制 |
|-----------|------|---------|
| WiFi 4 | 802.11n | 64-QAM |
| WiFi 5 | 802.11ac | 256-QAM |
| WiFi 6 | 802.11ax | 1024-QAM |
| WiFi 7 | 802.11be | 4096-QAM |

### QAM = ASK + PSK 的组合

QAM（正交幅度调制）同时调制幅度和相位：
- 16-QAM：16 个符号点（4 bit/symbol）
- 64-QAM：64 个符号点（6 bit/symbol）
- 256-QAM：256 个符号点（8 bit/symbol）
- 1024-QAM：1024 个符号点（10 bit/symbol）
- 4096-QAM：4096 个符号点（12 bit/symbol）

## 与验证工作的关联

### 验证关注点

| 验证项 | 说明 |
|--------|------|
| 符号映射 | 比特流是否正确映射到星座图 |
| 星座图精度 | EVM（误差矢量幅度）是否达标 |
| 调制切换 | 不同 MCS 切换是否正确 |
| 边界条件 | 极端信噪比下的解调性能 |

### EVM 与调制阶数的关系

调制阶数越高，对 EVM 要求越严格：

| 调制 | 目标 EVM |
|------|---------|
| BPSK | < -5 dB |
| QPSK | < -10 dB |
| 16-QAM | < -16 dB |
| 64-QAM | < -22 dB |
| 256-QAM | < -28 dB |
| 1024-QAM | < -34 dB |
| 4096-QAM | < -40 dB |
