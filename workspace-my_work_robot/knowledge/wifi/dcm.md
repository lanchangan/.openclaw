# DCM 双载波调制

> Dual Carrier Modulation - 通过符号复制实现频率分集

---

## 一句话总结

> **DCM 将同一个符号复制到两个间隔较远的子载波上，通过频率分集对抗深度衰落，提高传输可靠性。**

---

## 核心原理

### 普通调制 vs DCM

```
普通传输：
子载波1: [符号A] ← 如果这个子载波衰落，符号A就丢了

DCM传输：
子载波1: [符号A]  \
                    } 两个子载波传同一个符号
子载波N: [符号A]  /  即使一个衰落，另一个还能收到
```

### 关键特性

| 特性 | 说明 |
|------|------|
| **频率分集** | 两个子载波间隔足够大，不会同时衰落 |
| **带宽牺牲** | 有效数据速率减半 |
| **可靠性提升** | 适合低 SNR 或高可靠性场景 |

---

## DCM 支持的调制方式

> **DCM 不是独立的调制方式，而是一种符号复制技术**

DCM 可应用于：

| 调制方式 | 对应 MCS (802.11ax) |
|---------|---------------------|
| BPSK | MCS 0 |
| QPSK | MCS 1 |
| 16-QAM | MCS 3, 4 |

---

## 802.11ax 中的 DCM

| 原始 MCS | 调制 | 码率 | DCM 可用 |
|---------|------|------|---------|
| MCS 0 | BPSK | 1/2 | ✅ |
| MCS 1 | QPSK | 1/2 | ✅ |
| MCS 3 | 16-QAM | 1/2 | ✅ |
| MCS 4 | 16-QAM | 3/4 | ✅ |

---

## 802.11be (EHT) 中的 DCM

### MCS 索引

| MCS Index | 模式 |
|-----------|------|
| MCS 12 | 4096-QAM, 3/4 |
| MCS 13 | 4096-QAM, 5/6 |
| MCS 14 | EHT-DUP（重复传输模式） |
| **MCS 15** | **DCM**（双载波调制） |

---

## DCM vs EHT-DUP

| 特性 | DCM | EHT-DUP |
|------|-----|---------|
| **应用** | 同一符号在两个子载波重复 | 整个数据在多个频段重复 |
| **带宽效率** | 降低一半 | 降低更多 |
| **可靠性** | 高 | 更高 |
| **调制支持** | BPSK/QPSK/16-QAM | 任意调制 |
| **适用场景** | 低速高可靠 | 极端环境 |

---

## 与验证工作的关联

### 验证关注点

| 验证项 | 说明 |
|--------|------|
| **符号复制** | 同一个符号是否正确复制到两个子载波 |
| **子载波间隔** | 两个复制符号的子载波间隔是否足够大 |
| **解调合并** | 接收端如何合并两个子载波的信号 |
| **EVM** | DCM 模式下 EVM 是否有改善 |

### 测试场景

1. **正常场景**：两个子载波都正常接收
2. **单子载波衰落**：一个子载波深度衰落
3. **切换场景**：DCM 与非 DCM 模式切换
4. **边界条件**：不同调制方式下的 DCM 行为
