# 咸鱼发闲置 - 发布流程

## 整体流程

在咸鱼网页版（https://www.goofish.com/publish）发布闲置商品，遵循以下步骤：

## 步骤1：准备商品首图

### 对于虚拟商品（教程、网课等）：
1. 创建SVG格式图片，深色背景+白色文字，类似Claude Code官方风格
2. SVG模板：
```svg
<svg width="800" height="600" xmlns="http://www.w3.org/2000/svg">
  <rect width="100%" height="100%" fill="#1a1a2e"/>
  <text x="400" y="250" font-family="Arial, sans-serif" font-size="72" font-weight="bold" fill="#ffffff" text-anchor="middle">Claude Code</text>
  <text x="400" y="330" font-family="Arial, sans-serif" font-size="36" fill="#a8a8d8" text-anchor="middle">零基础实战教程</text>
  <rect x="100" y="400" width="600" height="4" fill="#00d4ff" opacity="0.6"/>
</svg>
```
3. 需要将SVG转换为JPG格式上传（可以手动浏览器打开截图保存JPG）

## 步骤2：填写表单信息

### 宝贝描述（建议50-100字）
模板：
```
[商品名称]，从入门到精通，包含完整案例源码，常用prompt合集和技巧总结。虚拟商品自动发货，拍下秒发链接，24小时在线，描述不符支持全额退款！
```

### 价格填写
- **价格**：填写你的售价（比如 1.5）
- **原价**：填写原价（比如 5）

### 发货设置
- **虚拟商品**：一定要选择 `无需邮寄`
- **宝贝所在地**：可以留空或者保持默认

## 步骤3：上传图片并发布

1. 点击 `添加首图` 按钮
2. 选择生成好的JPG格式图片上传
3. 检查所有信息填写正确
4. 点击 `发布` 按钮完成发布

## 步骤4：发布后检查

- 确认商品成功发布
- 如果是虚拟商品，可以在商品描述或者对话中发送百度网盘链接给买家

## 常用参数参考

| 商品类型 | 价格范围 | 原价 |
|---------|---------|------|
| 小型教程 | 1.5-5.0 | 5-30 |
| 完整课程 | 9.9-19.9 | 30-99 |

## 注意事项

1. 宝贝图片必须上传（必填项），否则无法发布
2. 虚拟商品必须选择"无需邮寄"
3. 百度网盘链接不要直接写在商品描述里，等买家拍下后再发
4. 价格填对，原价填写市场价格显示优惠
