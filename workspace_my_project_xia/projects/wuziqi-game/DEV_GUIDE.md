## 第一阶段：账号注册

### 步骤 1.1：进入注册页面

1. 打开浏览器，访问 **微信公众平台**：https://mp.weixin.qq.com
2. 点击页面右上角 **「立即注册」** 按钮
3. 在账号类型选择页面，点击 **「小程序」** 图标

```
直达链接：https://mp.weixin.qq.com/cgi-bin/registermidpage?action=index&lang=zh_CN
```

### 步骤 1.2：填写账号信息

| 字段 | 填写要求 |
|------|----------|
| **邮箱** | 未绑定过微信公众平台的邮箱（建议使用新邮箱） |
| **密码** | 8-20位，包含大小写字母、数字、特殊符号 |
| **验证码** | 点击获取，输入邮箱收到的验证码 |

**操作步骤**：
1. 输入邮箱地址
2. 设置登录密码
3. 点击「获取验证码」
4. 登录邮箱查收验证码（检查垃圾邮件箱）
5. 输入验证码
6. 勾选「我同意《微信公众平台服务协议》」
7. 点击「注册」按钮

### 步骤 1.3：邮箱激活

1. 登录注册时填写的邮箱
2. 找到微信团队发送的激活邮件（发件人：weixin_mp@tencent.com）
3. 点击邮件中的激活链接
4. 页面跳转提示「激活成功」

> ⚠️ 如果收件箱没有邮件，请检查垃圾邮件文件夹

### 步骤 1.4：选择主体类型

**个人开发者操作**：

1. 在主体类型选择页面，选择 **「个人」**
2. 点击「下一步」

### 步骤 1.5：填写主体信息（个人）

| 字段 | 填写内容 |
|------|----------|
| **管理员姓名** | 本人真实姓名 |
| **管理员身份证号** | 本人身份证号码 |
| **身份证有效期** | 选择身份证上的有效期 |
| **身份证正反面照片** | 上传清晰照片（支持jpg/png，每张不超过2MB） |
| **管理员手机号** | 本人手机号 |
| **短信验证码** | 点击获取，输入手机收到的验证码 |

**操作步骤**：
1. 填写姓名和身份证号
2. 选择身份证有效期
3. 上传身份证正面照片（人像面）
4. 上传身份证反面照片（国徽面）
5. 填写手机号并获取验证码
6. 输入验证码
7. 使用管理员微信扫描页面上的二维码
8. 确认管理员身份
9. 点击「提交」

### 步骤 1.6：等待审核

- 个人主体注册通常 **即时通过** 或几分钟内完成
- 审核通过后会收到微信团队通知

### 步骤 1.7：获取 AppID

注册成功后：

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 使用管理员微信扫码登录
3. 进入「开发管理」→「开发设置」
4. 在页面顶部找到 **AppID（小程序ID）**
5. 复制保存（格式类似：wx1234567890abcdef）

> 📌 AppID 是小程序的唯一标识，创建项目时必须使用

## 第二阶段：开发环境搭建

### 步骤 2.1：下载开发者工具

1. 访问下载页面：https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html
2. 根据操作系统选择版本：
   - Windows 64位：选择「Windows 64」下载
   - Windows 32位：选择「Windows 32」下载
   - macOS Intel：选择「macOS Intel」下载
   - macOS M1/M2：选择「macOS Apple Silicon」下载
3. 下载完成后运行安装程序
4. 按提示完成安装（建议安装到非系统盘）

### 步骤 2.2：登录开发者工具

1. 启动「微信开发者工具」
2. 使用管理员微信扫描登录二维码
3. 登录成功后进入项目列表页面

### 步骤 2.3：创建新项目

1. 点击左侧「小程序」标签
2. 点击 **「+」** 号（或「新建项目」按钮）
3. 填写项目信息：

| 字段 | 填写内容 |
|------|----------|
| **项目名称** | 五子棋小游戏（或自定义名称） |
| **目录** | 选择项目存放位置（如：D:\projects\wuziqi） |
| **AppID** | 输入步骤1.7获取的AppID（或选择「测试号」先体验） |
| **开发模式** | 选择「小程序」 |
| **后端服务** | 选择「不使用云服务」或「微信云开发」 |

4. 点击「新建」按钮
5. 项目创建成功，自动进入开发界面

### 步骤 2.4：熟悉开发界面

## 第三阶段：小程序开发

### 步骤 3.1：了解项目结构

创建项目后，默认生成以下结构：

```
wuziqi/
├── pages/                    # 页面文件夹
│   └── index/               # 首页
│       ├── index.wxml       # 页面结构（类似HTML）
│       ├── index.wxss       # 页面样式（类似CSS）
│       ├── index.js         # 页面逻辑（JavaScript）
│       └── index.json       # 页面配置
├── utils/                    # 工具函数
│   └── util.js
├── app.js                    # 小程序入口文件
├── app.json                  # 全局配置
├── app.wxss                  # 全局样式
├── project.config.json       # 项目配置
└── sitemap.json              # 小程序索引配置
```

### 步骤 3.2：配置小程序信息

**编辑 app.json**：

```json
{
  "pages": [
    "pages/index/index",
    "pages/game/game"
  ],
  "window": {
    "navigationBarTitleText": "五子棋",
    "navigationBarBackgroundColor": "#1a1a2e",
    "navigationBarTextStyle": "white",
    "backgroundColor": "#16213e",
    "backgroundTextStyle": "dark"
  },
  "style": "v2",
  "sitemapLocation": "sitemap.json"
}
```

### 步骤 3.3：创建游戏页面

1. 在 `pages` 目录下创建 `game` 文件夹
2. 创建以下文件：
   - game.wxml
   - game.wxss
   - game.js
   - game.json

### 步骤 3.4：开发核心功能

**五子棋核心模块**：

| 模块 | 文件 | 功能 |
|------|------|------|
| 棋盘绘制 | game.js | Canvas 绘制15x15棋盘 |
| 落子交互 | game.js | 触摸事件处理 |
| 胜负判定 | utils/win-check.js | 五子连珠检测 |
| AI算法 | utils/ai.js | 电脑对战逻辑 |

### 步骤 3.5：本地调试

1. 在开发者工具中点击「编译」按钮
2. 在模拟器中查看效果
3. 使用 Console 面板调试 JavaScript
4. 使用 AppData 面板查看数据状态
5. 使用 Network 面板查看网络请求

### 步骤 3.6：真机调试

1. 点击「预览」按钮
2. 使用手机微信扫描二维码
3. 在真机上测试小程序
4. 或点击「真机调试」进行远程调试

## 第四阶段：小程序备案

> ⚠️ **重要**：自2023年9月1日起，所有小程序必须完成备案才能上架

### 步骤 4.1：准备备案材料

**个人主体所需材料**：

| 材料 | 要求 |
|------|------|
| 身份证正反面照片 | 清晰、完整、在有效期内 |
| 手机号码 | 用于接收核验短信 |
| 小程序名称 | 确定最终名称（备案后不可随意更改） |
| 小程序简介 | 描述小程序功能用途 |
| 服务类目 | 选择与功能匹配的类目 |

**避坑提示**：
- ❌ 名称避免商业词汇：预约、购物、商城、火锅、酒店等
- ✅ 推荐使用工具类名称：XX学习工具、XX助手、XX小游戏

### 步骤 4.2：进入备案页面

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入「设置」→「基本设置」
3. 找到「小程序备案」模块
4. 点击「去备案」

### 步骤 4.3：填写备案信息

**环节一：主体信息填写**

| 字段 | 填写内容 |
|------|----------|
| 主办单位名称 | 本人姓名 |
| 证件类型 | 居民身份证 |
| 证件号码 | 身份证号 |
| 证件住所 | 身份证上的地址 |
| 证件有效期 | 身份证有效期 |
| 负责人姓名 | 本人姓名 |
| 负责人证件号 | 身份证号 |
| 负责人手机 | 本人手机号 |
| 办公电话 | 可选 |
| 电子邮箱 | 常用邮箱 |
| 通讯地址 | 现居住地址 |

**环节二：小程序信息填写**

| 字段 | 填写内容 |
|------|----------|
| 小程序名称 | 确定后的名称 |
| 小程序简介 | 描述功能用途 |
| 服务类目 | 选择匹配的类目 |
| 服务内容标识 | 根据类目选择 |

**环节三：上传材料**

- 上传身份证正反面
- 上传手持身份证照片（部分情况需要）
- 上传小程序截图

### 步骤 4.4：平台初审

1. 提交备案后，腾讯平台进行初审
2. 初审时间：**1-2个工作日**
3. 审核结果通过站内信、模板消息通知
4. ⚠️ 保持电话畅通，可能需要电话核验

**常见初审问题**：
- 名称不合规 → 修改名称重新提交
- 类目不匹配 → 重新选择类目
- 材料不清晰 → 重新上传材料

### 步骤 4.5：工信部短信核验

1. 初审通过后，收到工信部短信（发送号码：12381）
2. 短信内容包含验证码
3. **24小时内**完成核验，否则自动驳回

**核验步骤**：
1. 访问工信部备案系统：https://beian.miit.gov.cn
2. 点击「短信核验」
3. 输入手机号和验证码
4. 点击「验证」完成核验

### 步骤 4.6：通管局审核

1. 短信核验成功后，进入通管局审核
2. 审核时间：**1-20个工作日**（各地不同）
3. 审核结果通过站内信、模板消息、短信通知

### 步骤 4.7：备案成功

- 收到备案成功通知
- 在小程序后台可查看备案号
- 可以上传代码提交审核

**备案周期总结**：

```
提交备案 → 平台初审(1-2天) → 短信核验(24小时内) → 通管局审核(1-20天) → 备案成功

总周期：约3-23个工作日
```

## 第五阶段：小程序认证

> 个人小程序认证费用：**30元**
> 认证后才能被搜索、分享

### 步骤 5.1：进入认证页面

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入「设置」→「基本设置」
3. 找到「微信认证」模块
4. 点击「去认证」

### 步骤 5.2：填写认证信息

| 字段 | 填写内容 |
|------|----------|
| 认证类型 | 个人 |
| 认证名称 | 小程序名称 |
| 认证说明 | 描述小程序用途 |

### 步骤 5.3：支付认证费用

1. 确认信息无误
2. 使用微信扫码支付 **30元**
3. 支付成功后等待审核

### 步骤 5.4：等待认证结果

- 认证时间：**1-3个工作日**
- 认证成功后收到微信通知
- ⚠️ 认证失败费用不退还

## 第六阶段：代码上传与发布

### 步骤 6.1：上传代码

1. 在微信开发者工具中打开项目
2. 点击右上角「上传」按钮
3. 填写版本号（如：1.0.0）
4. 填写项目备注（如：初始版本，包含基础功能）
5. 点击「上传」
6. 等待上传完成

### 步骤 6.2：提交审核

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入「管理」→「版本管理」
3. 在「开发版本」列表中找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息：

| 字段 | 填写内容 |
|------|----------|
| 功能页面类目 | 选择与小程序匹配的类目 |
| 功能页面标签 | 添加相关标签 |
| 测试账号 | （如需要）提供测试账号密码 |

6. 确认提交

### 步骤 6.3：等待审核

- 审核时间：通常 **1个工作日内**
- 审核结果通过站内信、模板消息通知

### 步骤 6.4：发布上线

1. 审核通过后，收到通知
2. 登录微信公众平台
3. 进入「管理」→「版本管理」
4. 在「审核版本」中点击「发布」
5. 选择发布方式：
   - 全量发布：所有用户立即看到新版本
   - 灰度发布：分批发布给用户
6. 确认发布
7. 小程序正式上线！

## 常见问题 FAQ

### Q1：个人小程序有什么限制？

| 功能 | 个人主体 | 企业主体 |

### Q2：备案期间可以开发吗？

可以。备案和开发可以并行进行，建议：
- 注册完成后立即提交备案
- 备案审核期间进行开发
- 备案通过后即可上传代码

### Q3：名称被拒绝怎么办？

1. 查看拒绝原因（通常含商业词汇）
2. 修改名称，避免：预约、商城、酒店等词
3. 使用工具类名称：XX助手、XX工具、XX小游戏
4. 重新提交

### Q4：身份证地址和现居住地不一致怎么办？

提供以下材料之一：
- 居住证或暂住证
- 社保缴纳证明（可通过微信小程序查询下载PDF）
