## 三、五子棋项目功能规划

### 核心功能

| 功能模块 | 优先级 | 说明 |
|----------|--------|------|
| 棋盘绘制 | P0 | Canvas绘制15x15棋盘 |
| 落子交互 | P0 | 点击交叉点落子 |
| 胜负判定 | P0 | 五子连珠检测 |
| 回合切换 | P0 | 黑白棋交替下子 |
| 游戏重置 | P0 | 重新开始功能 |

### 进阶功能

| 功能模块 | 优先级 | 说明 |
|----------|--------|------|
| AI对战 | P1 | 单人对战电脑 |
| 悔棋功能 | P1 | 撤销上一步 |
| 游戏记录 | P1 | 记录对局历史 |
| 在线对战 | P2 | 双人联机（需云开发） |
| 排行榜 | P2 | 战绩排名（需云开发） |

### AI算法方案

```
评分策略：
- 遍历棋盘所有空位
- 计算每个位置的进攻分值和防守分值
- 选择分值最高的位置落子

评分规则：
- 1子连珠：玩家10分 / AI15分
- 2子连珠：玩家20分 / AI25分
- 3子连珠：玩家40分 / AI45分
- 4子连珠：玩家80分 / AI85分
```

## 六、注意事项

### 个人主体限制

- ❌ 无法使用支付功能
- ❌ 无法开通微信支付
- ✅ 可以接入广告变现
- ✅ 可以使用云开发

### 开发注意事项

1. **Canvas适配**：不同设备屏幕尺寸需做适配
2. **性能优化**：避免频繁重绘棋盘
3. **触摸事件**：正确处理touchstart/touchend
4. **AI优化**：基础AI防守偏重，可后续优化进攻策略
