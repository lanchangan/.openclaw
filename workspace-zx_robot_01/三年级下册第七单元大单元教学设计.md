# 统编版小学语文三年级下册第七单元大单元教学设计
## 单元主题：探索奇妙世界，发现自然之美

---

## 一、大单元设计理念与理论依据

### （一）设计理念
基于2022年版义务教育语文课程标准"学习任务群"理念，本单元设计坚持**核心素养导向**，以"大概念"统领单元学习，通过**任务驱动**整合阅读、鉴赏、表达、交流、探究等语文实践活动，实现"教-学-评"一致性，促进学生语言运用、思维能力、审美创造和文化自信的全面发展。

### （二）理论框架
本设计遵循大单元教学的基本操作框架：
1. **主题引领**：确立"探索奇妙世界，发现自然之美"的单元大主题
2. **任务驱动**：设计一个贯穿单元的核心大任务"我的奇妙自然手册"
3. **活动进阶**：搭建"感知奇妙-探究奇妙-表达奇妙-分享奇妙"的学习进阶
4. **真实输出**：最终产出可视化学习成果，实现知识迁移与运用
5. **评价一体化**：设计与目标匹配的过程性评价和终结性评价

## 二、单元教材分析

### （一）单元整体说明
本单元是统编版小学语文三年级下册第七单元，新版教材编排了三篇精读课文：
- **《我们奇妙的世界》**：总写世界的奇妙，从天空和大地两个方面展现奇妙景象
- **《暴风雨来临之前》**：细致描绘暴风雨来临前自然界的奇特变化
- **《火烧云》**：生动刻画火烧云颜色和形状的瞬息万变

单元导语：**"天地间隐藏着无穷无尽的奥秘，等待我们去寻找"**，点明了单元学习方向——引导学生留心观察，发现世界的奇妙。

### （二）语文要素
本单元核心语文要素：**"仔细观察，发挥想象，把句子写生动"**
- 阅读方面：学习作者是怎样细致观察事物，用生动语言描写变化的
- 习作方面：引导学生观察身边事物，写清楚事物的变化，发挥想象写生动

### （三）教材变化说明
与旧版相比，新版教材替换了《海底世界》为《暴风雨来临之前》，更突出了"变化"这一核心，三篇文章都聚焦自然现象的动态变化，内在逻辑更加统一：
- 《我们奇妙的世界》：整体感知世界的奇妙（静态+动态）
- 《暴风雨来临之前》：聚焦特定时刻自然事物的变化过程
- 《火烧云》：聚焦一种自然景观的瞬息变化

## 三、单元教学目标

### （一）核心素养目标

| 核心素养 | 具体目标 |
|---------|---------|
| **文化自信** | 感受大自然的神奇美丽，培养热爱自然、探索自然的情感 |
| **语言运用** | 1. 认识31个生字，会写28个生字，积累课文中优美的语句<br>2. 能正确、流利、有感情地朗读课文，背诵精彩段落<br>3. 能把观察到的事物变化写清楚，发挥想象写生动 |
| **思维能力** | 1. 学习作者多感官观察的方法，发展有序观察和捕捉变化的能力<br>2. 通过想象画面，培养形象思维能力<br>3. 比较三篇课文描写变化的不同方法，发展比较归纳思维 |
| **审美创造** | 1. 感受作者笔下自然景观的美，培养审美敏感度<br>2. 能用生动语言表达自己发现的美，尝试用图文结合方式展现美 |

### （二）目标分层
- **低阶目标**：掌握生字词，能正确朗读课文，理解课文内容
- **高阶目标**：运用学到的观察方法，完成"我的奇妙自然手册"创作，解决真实情境下的观察表达问题

## 四、单元整体规划

### （一）核心大任务
**"制作我的奇妙自然手册"**
> 情境创设：学校要举办"奇妙自然"摄影征文展，请你用图文结合的方式，制作一本属于自己的"奇妙自然手册"，分享你发现的奇妙世界。

### （二）学习任务群架构

| 学习任务 | 课时安排 | 语文实践活动 |
|---------|---------|-------------|
| **任务一：开启奇妙之旅——单元启动课** | 1课时 | 1. 单元导语导入，交流"我发现的奇妙现象"<br>2. 浏览单元课文，整体感知单元主题<br>3. 明确核心任务，制定学习计划 |
| **任务二：感知奇妙世界——文本阅读探究** | 6课时 | 《我们奇妙的世界》：2课时<br>《暴风雨来临之前》：2课时<br>《火烧云》：2课时 |
| **任务三：学习表达方法——语文园地** | 2课时 | 1. 交流平台：总结观察和表达的方法<br>2. 词句段运用：积累动态描写的语句，练习写变化<br>3. 日积月累：积累古诗《滁州西涧》 |
| **任务四：实践探究发现——观察实践活动** | 1课时 + 课外 | 1. 学习观察计划制定方法<br>2. 开展课外观察实践，记录变化 |
| **任务五：成果创作展示——习作与分享** | 2课时 | 1. 完成习作"我的奇妙发现"<br>2. 整理制作"奇妙自然手册"<br>3. 班级分享展示 |

**总课时：12课时**

## 五、分课时教学设计

### 任务一：开启奇妙之旅——单元启动课（1课时）

#### 教学目标：
1. 初步感知单元主题，激发探索兴趣
2. 明确单元核心任务，整体规划学习

#### 教学过程：
1. **激趣导入**（5分钟）
   - 播放一段自然风光短视频，提问：你看到了哪些奇妙的景象？
   - 板书单元主题：探索奇妙世界

2. **交流分享**（10分钟）
   - 提问：在生活中，你发现过哪些奇妙的自然现象？
   - 小组交流，全班分享

3. **浏览课文**（10分钟）
   - 学生快速浏览三篇课文，看看作者发现了哪些奇妙
   - 填写"单元预习单"：
     - 课文题目：______，写了______的奇妙
     - 我最感兴趣的是：______

4. **明确任务**（10分钟）
   - 介绍核心任务：制作"我的奇妙自然手册"
   - 手册内容：
     1. 精彩摘抄：摘抄课文中写得奇妙的语句
     2. 我的发现：写自己观察到的奇妙自然现象
     3. 美图欣赏：贴上相关照片或绘画
   - 讨论：我们需要哪些步骤完成这个任务？

5. **作业布置**（5分钟）
   - 预习三篇课文，自学生字词

### 任务二：文本阅读探究

#### 《我们奇妙的世界》（第一课时）

**教学目标：**
1. 认识生字，正确读写词语
2. 正确朗读课文，理清课文结构
3. 感受天空的奇妙，积累优美语句

**教学过程：**
1. **导入**（5分钟）
   - 复习单元主题，板书课题
   - 提问：读了课题，你想知道什么？

2. **初读感知**（15分钟）
   - 学生自由读课文，圈出生字词
   - 生字词检查：呈现"呈现、变幻、蜡烛、威武、凶猛"等词语，正音
   - 指名分段读课文，思考：课文是从哪几个方面写世界的奇妙？
   - 梳理结构：总起（1）→天空（2-8）→大地（9-16）→总结（17-18）

3. **探究天空的奇妙**（15分钟）
   - 默读课文2-8自然段，思考：作者写了天空哪些奇妙的景象？
   - 填写表格：
     | 时间/景象 | 有什么奇妙之处 |
     |-----------|----------------|
     | 日出 | 天空颜色从蓝色变成粉红色，再变成金色 |
     | 云彩 | 各种形状，像雕刻一样 |
     | 雨后 | 水洼像镜子，映出你的脸 |
     | 黄昏 | 云彩变成金色，像巨人一样 |
     | 黑夜 | 星星闪烁，像在眨眼 |
   - 读一读你觉得最奇妙的句子，说说为什么觉得奇妙

4. **总结积累**（5分钟）
   - 小结：作者按时间顺序写出了天空一天中的奇妙变化
   - 作业：背诵你喜欢的段落

#### 《我们奇妙的世界》（第二课时）

**教学目标：**
1. 感受大地的奇妙，学习作者按季节顺序描写的方法
2. 理解"世界上存在的奇妙事物是无穷的"这句话的含义
3. 积累优美语句

**教学过程：**
1. **复习导入**（5分钟）
   - 背诵精彩段落
   - 提问：上节课我们欣赏了天空的奇妙，这节课来看大地的奇妙

2. **探究大地的奇妙**（20分钟）
   - 默读9-16自然段，思考：作者写了大地哪些季节的奇妙？
   - 小组合作探究：每个季节有什么奇妙之处？
     - **春天**：植物生长，绿色树叶，种子生根发芽
     - **夏天**：大树绿荫，人们乘凉
     - **秋天**：树叶变黄，蝴蝶飞舞，鸟儿筑巢
     - **冬天：**冰雪融化，水滴变成小冰柱，像刀剑
   - 讨论：作者观察大地用了什么顺序？（季节顺序）
   - 品味语言：你最喜欢哪个句子？为什么？
     - 例："太阳就像一个大火球一样升起来了"——比喻生动
     - 例："水滴就像一颗颗珍珠"——形象写出水滴的晶莹

3. **总结全文**（10分钟）
   - 齐读最后两个自然段，理解"只要我们仔细地观察，寻找，我们就能从极普通的事物中找到美"
   - 讨论：联系生活，说说你从哪些极普通的事物中找到过美？
   - 总结写法：作者从天空到大地，按照时间顺序，用生动的比喻，写出了世界的奇妙

4. **作业**（5分钟）
   - 摘抄本课中你认为最奇妙的3句话，准备收入"奇妙自然手册"

---

#### 《暴风雨来临之前》（第一课时）

**教学目标：**
1. 认识生字新词，正确朗读课文
2. 理清课文顺序，了解课文写了暴风雨前哪些景物的变化

**教学过程：**
1. **导入**（5分钟）
   - 提问：你经历过暴风雨吗？暴风雨来临之前，周围有什么变化？
   - 板书课题，齐读

2. **初读课文**（15分钟）
   - 学生自由读，标出自然段，圈出生字词
   - 检查字词："狂飙、闷雷、湛蓝、涟漪、静谧、闷热"等，正音理解
   - 指名分段读课文，思考：课文是按照什么顺序写的？
     - （时间顺序：从闷热无风到狂风大作，暴风雨越来越近）

3. **整体梳理**（15分钟）
   - 默读课文，想一想：暴风雨来临之前，作者观察了哪些事物？它们有什么变化？
   - 完成填空：
     - 天空：原本______，后来______
     - 空气：原本______，后来______
     - 树叶：原本______，后来______
     - 鸟儿：原本______，后来______
   - 全班交流，梳理课文内容

4. **总结布置作业**（5分钟）
   - 小结：作者细致观察了暴风雨来临前各种事物的变化
   - 作业：画出你印象最深的变化场景，下节课交流

#### 《暴风雨来临之前》（第二课时）

**教学目标：**
1. 品味作者生动描写变化的语言
2. 学习作者多感官观察的方法
3. 感受大自然的神奇变幻

**教学过程：**
1. **复习导入**（5分钟）
   - 回顾：课文写了暴风雨前哪些事物的变化？
   - 交流：你画了哪个场景，为什么印象最深？

2. **品味语言**（20分钟）
   - 聚焦描写，合作探究：
     1. 作者用了哪些方法写天气闷热？
        - 感觉："像着了火一样"（触觉）
        - 景物："绿叶一动也不动"（视觉）
        - 动物："鱼儿都跑到水面上来透气"（动态）
     2. 第一阵风来了，哪些变化让你觉得最奇妙？
        - "一缕细风"→"树叶开始低语"→"风变大了"→"整个树林都喧嚣起来"
        - 写出了风从小到大的变化过程
     3. 作者为什么能把变化写得这么清楚？
        - （因为观察很细致，按顺序写，写出了变化过程）
   - 多感官体验：找找课文中从不同感官描写的句子
     - 视觉：天空颜色变化，树木摇动
     - 听觉：树叶声音，雷声
     - 触觉：空气闷热，风吹过来

3. **写法总结**（10分钟）
   - 讨论：和《我们奇妙的世界》比，这篇课文写法有什么不同？
     - 《我们奇妙的世界》：写不同景物，面面俱到
     - 《暴风雨来临之前》：集中写一个过程，一步步变化
   - 小结：写变化要按顺序，一步一步写出变化过程

4. **积累运用**（5分钟）
   - 摘抄生动描写变化的句子到你的手册中
   - 作业：想一想，你见过的天气变化，试着说一个变化过程

#### 《火烧云》（第一课时）

**教学目标：**
1. 认识生字，正确读写词语
2. 正确流利朗读课文，了解课文主要内容
3. 感受火烧云颜色变化的奇妙

**教学过程：**
1. **导入**（5分钟）
   - 出示火烧云图片，提问：你见过这样的云吗？它有什么特点？
   - 简介作者萧红，板书课题

2. **初读课文**（15分钟）
   - 学生自由读，自学生字词
   - 检查："紫檀色、威武、凶猛、镇静、一模一样"正音
   - 思考：课文写了火烧云哪些方面的变化？
     - （颜色变化和形状变化）

3. **探究颜色变化**（15分钟）
   - 读课文1-3段，思考：火烧云颜色变化有什么特点？
   - 找出表示颜色的词语：
     - 紫檀色、红彤彤、金灿灿、半紫半黄、半灰半百合色
     - 葡萄灰、梨黄、茄子紫
   - 讨论：
     - 颜色变化快还是慢？（变化极快）——"一会儿"
     - 颜色多不多？（多种多样，还有很多说不出来的）
   - 仿写练习：照样子说颜色
     - 红彤彤：______ ______ ______
     - 葡萄灰：______ ______ ______
     - 半紫半黄：______ ______ ______

4. **总结作业**（5分钟）
   - 小结：作者用丰富的词语写出了火烧云颜色多、变化快的特点
   - 作业：背诵1-3自然段

#### 《火烧云》（第二课时）

**教学目标：**
1. 感受火烧云形状变化的奇妙
2. 学习作者写变化的方法
3. 有感情朗读课文，积累优美语言

**教学过程：**
1. **复习导入**（5分钟）
   - 背诵1-3段
   - 提问：火烧云颜色有什么特点？这节课我们来看形状变化

2. **探究形状变化**（20分钟）
   - 默读4-7段，思考：火烧云形状变化有什么特点？
   - 梳理：作者写了哪几种形状？
     - 一匹马→大狗→大狮子
   - 填表：
     | 形状 | 出现了 | 变化了 | 不见了 |
     |------|--------|--------|--------|
     | 马 | 跪着，准备起来 | 腿伸开了，尾巴不见了 | 变模糊了 |
     | 狗 | 跑着，后面跟着小狗 |  | 不见了 |
     | 狮子 | 很威武，很镇静 |  | 渐渐变模糊了 |
   - 讨论：形状变化有什么特点？（变化快，一会儿就不见了）
   - 品味语言："忽然又来了一条大狗""可是一转眼就变了"——写出变化之快

3. **总结全文比较阅读**（10分钟）
   - 齐读最后一段"一会儿工夫，火烧云下去了"
   - 比较三篇课文写变化：
     - 《我们奇妙的世界》：一天四季的缓慢变化
     - 《暴风雨来临之前》：几十分钟的过程变化
     - 《火烧云》：几分钟的瞬息变化
   - 小结：不管是哪种变化，都要按照顺序，写出变化过程
