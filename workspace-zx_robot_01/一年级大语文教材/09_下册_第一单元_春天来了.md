# 一年级下册 第一单元 春天来了

## 单元主题
带领学生走进春天，感受春天的勃勃生机，观察春天的变化，培养热爱大自然、热爱生活的美好情感。

## 语文要素
1. 认识"春、暖、花、开、燕、柳"等30个常用汉字
2. 会写"春、风、花、草、树、木"等10个生字
3. 能正确、流利、有感情地朗读课文，背诵关于春天的古诗和优美段落
4. 能按顺序观察春天的景物，用几句话描述春天的特点
5. 学习积累描写春天的词语和句子

## 文化要素
1. 了解二十四节气中的"立春"、"雨水"、"惊蛰"、"春分"四个春季节气
2. 学习古人咏春的诗词，感受春天的美好
3. 了解春天的传统习俗，如春游、放风筝、植树等

## 一、课文板块

### 课文1：找春天
春天来了！春天来了！
我们几个孩子脱掉棉袄，冲出家门，奔向田野，去寻找春天。
春天像个害羞的小姑娘，遮遮掩掩，躲躲藏藏。我们仔细地找哇，找哇。
小草从地下探出头来，那是春天的眉毛吧？
早开的野花一朵两朵，那是春天的眼睛吧？
树木吐出点点嫩芽，那是春天的音符吧？
解冻的小溪叮叮咚咚，那是春天的琴声吧？
春天来了！我们看到了她，我们听到了她，我们闻到了她，我们触到了她。她在柳枝上荡秋千，在风筝尾巴上摇哇摇；她在喜鹊、杜鹃嘴里叫，在桃花、杏花枝头笑……

**生字表**：
会认：脱、袄、冲、奔、寻、羞、遮、掩、躲、藏、嫩、符、触、鹃
会写：春、冬、风、雪、花、飞、入、姓、什、么

**课文讲解**：
春天是个充满生机的季节，作者用生动的比喻，把春天写得活灵活现。让我们一起走出家门，到大自然中去寻找春天的足迹吧！

### 课文2：咏柳
（唐）贺知章
碧玉妆成一树高，
万条垂下绿丝绦。
不知细叶谁裁出，
二月春风似剪刀。

**生字表**：
会认：咏、妆、绦、裁、剪
会写：诗、村、童、碧、妆、绿、丝、剪

**课文讲解**：
这首诗把柳树比作用碧玉打扮成的美女，把春风比作剪刀，形象地写出了春天柳树的美丽。春风真是个神奇的裁剪师，剪出了细细的柳叶，剪出了美丽的春天。

### 课文3：开满鲜花的小路
鼹鼠先生收到长颈鹿大叔寄来的一个包裹。他打开包裹一看，里面是一堆小颗粒，认不出是什么东西。
鼹鼠先生赶紧骑着摩托车，到松鼠太太家去。可是，他走得太急了，包裹破了，里面的东西都漏在来时的路上了。
春天来了，鼹鼠先生路过刺猬太太家，看到门前开着一大片绚丽多彩的鲜花。刺猬太太惊奇地说："这是谁在我家门前种的花？多美啊！"
鼹鼠先生回答："我不知道！"
鼹鼠先生路过狐狸太太家，看到门前也开着一大片五颜六色的鲜花。狐狸太太奇怪地问："这是谁在我家门前种的花？真美啊！"
鼹鼠先生回答："我不知道！"
鼹鼠先生来到松鼠太太家。松鼠太太走出门，看见门前的小路上花朵簇簇，小松鼠、小刺猬和小狐狸在那里快活地蹦啊跳啊。
松鼠太太对鼹鼠先生说："我知道了，去年长颈鹿大叔寄给你的是花籽。这是多么美好的礼物啊！"

**生字表**：
会认：邮、递、裹、寄、局、堆、破、漏、懊、丧、啊、猬、绚、籽
会写：鲜、邮、递、员、原、叔、局、堆、礼、车

**课文讲解**：
不小心漏在路上的花籽，第二年春天长成了开满鲜花的小路，给大家带来了快乐。美好的事物大家一起分享，会让更多的人感到快乐。

### 课文4：邓小平爷爷植树
1987年4月5日，是个令人难忘的日子。
这一天，碧空如洗，万里无云。在北京天坛公园植树的人群里，83岁高龄的邓小平爷爷格外引人注目。只见他手握铁锹，兴致勃勃地挖着树坑，额头已经满是汗珠，仍不肯休息。
一个树坑挖好了。邓爷爷精心地挑选了一棵茁壮的柏树苗，小心地移入树坑，又挥锹填了几锹土。他站到几步之外仔细看看，觉得不很直，连声说："不行，不行！"他又走上前把树苗扶正。
一棵绿油油的小柏树栽好了，就像战士一样笔直地站在那里。邓爷爷的脸上露出了满意的笑容。
今天，邓小平爷爷亲手栽种的柏树已经长大了，成了天坛公园一处美丽的风景。

**生字表**：
会认：邓、坛、龄、格、握、挖、选、茁、移、挥、填、扶、亭、咨、询、剧、餐
会写：邓、植、格、引、注、满、休、息、林、爷

## 二、阅读材料

### 阅读1：春妈妈的三个小姑娘
春妈妈回来了，
带来了三个淘气的小姑娘。
雷姑娘天天敲着鼓儿玩，
敲醒了天空，
敲醒了田野，
敲醒了沉睡的山岭和村庄。
雨姑娘天天洒着水儿玩，
洗绿了大树，
洗青了小草，
洗红了花朵的脸儿一张张。
风姑娘天天吹着气儿玩，
吹柔了树枝，
吹长了青草，
吹得春天朝前长。
啊，春天来了，
春妈妈带来了三个能干的小姑娘。

**阅读提示**：
春妈妈的三个小姑娘多么可爱呀！她们让大地变得生机勃勃，给我们带来了美丽的春天。

---

### 阅读2：孔子游春
春天到了，孔子带着弟子们到泗水河边游玩。
阳光暖暖地照着，泗水河边桃红柳绿，草色青青，习习的春风像优美的琴声，在给翩翩到来的春天伴奏。
孔子看着奔流不息的泗水，对弟子们说："水奔流不息，是哺育一切生灵的乳汁，它好像有德行。水没有一定的形状，或方或长，流必向下，和顺温柔，它好像有情义。水穿山岩，凿石壁，从无惧色，它好像有志向。万物入水，必能荡涤污垢，它好像善施教化……由此看来，水是真君子啊！"

弟子们听了老师的话，都明白了做人的道理。

**阅读提示**：
孔子从流水中悟出了做人的道理，我们也要像孔子一样，多观察，多思考，从大自然中学到更多的知识。

---

### 阅读3：植树的好处
春天是植树的好季节。植树的好处可多啦：
1. 树木可以净化空气，让我们呼吸到新鲜的空气
2. 树木可以防风固沙，减少自然灾害
3. 树木可以调节气候，让夏天更凉快
4. 树木可以美化环境，让我们的家乡更美丽
5. 树木长大了还可以用来做家具、盖房子

每年的3月12日是我们国家的植树节，这一天大家都要参加植树活动，为绿化祖国贡献自己的一份力量。

## 三、练习活动

### 活动1：春天的发现
**目标**：培养观察能力和表达能力
**过程**：
1. 带领学生到校园里或者公园里寻找春天
2. 让学生说说自己发现了哪些春天的变化，比如小草绿了，花开了，燕子飞回来了
3. 把自己的发现画下来，或者用几句话写下来
4. 上台分享自己的发现，看看谁的发现最多，最有趣

### 活动2：春天词语大聚会
**目标**：丰富词汇积累
**过程**：
1. 让学生说说自己知道的描写春天的词语，比如"春暖花开"、"春光明媚"、"桃红柳绿"
2. 把大家说的词语写在黑板上，一起读一读
3. 进行词语比赛，看谁说的词语最多
4. 选择几个自己喜欢的词语，说一句话

### 活动3：我们去春游
**目标**：培养规划能力和口语表达能力
**过程**：
1. 讨论：如果我们要去春游，需要做哪些准备？要注意哪些问题？
2. 分小组制定春游计划，包括去哪里、带什么东西、做什么活动、要遵守什么规则
3. 每个小组派代表介绍自己的春游计划
4. 大家一起评选出最好的春游计划

## 四、拓展任务

### 任务1：我的植物朋友
**跨学科融合**：语文 + 科学 + 观察
**要求**：
1. 选择一种春天的植物，比如迎春花、柳树、桃花
2. 连续观察一周，记录下它的变化，比如什么时候发芽，什么时候开花
3. 可以画下来，也可以拍照记录
4. 写几句话介绍你的植物朋友，说说它的特点，你为什么喜欢它

### 任务2：我和春天有个约会
**跨学科融合**：语文 + 美术 + 实践
**要求**：
1. 和爸爸妈妈一起去郊外春游，感受春天的美好
2. 可以放风筝、挖野菜、拍照片
3. 画一幅"春天的图画"，或者做一个春天的标本
4. 把你的作品带到学校，和同学们分享你春游的快乐

### 任务3：我是植树小能手
**跨学科融合**：语文 + 劳动 + 环保教育
**要求**：
1. 在爸爸妈妈的帮助下，种一棵小树苗，或者在家里种一盆花
2. 学习如何种树，如何照顾小树苗
3. 定期给小树浇水、施肥，观察它的生长情况
4. 写一篇观察日记，记录小树的生长过程

## 五、文化延伸

### 传统文化：春天的节气
春天有六个节气，我们已经学过了，它们是：
1. **立春**：春天的开始，天气开始变暖
2. **雨水**：降水增多，小雨淅淅沥沥
3. **惊蛰**：春雷惊醒了冬眠的小动物
4. **春分**：白天和黑夜一样长
5. **清明**：天气晴朗，草木繁茂，人们会去扫墓、踏青
6. **谷雨**：雨水增多，有利于谷物生长

每个节气都有不同的特点和习俗，比如惊蛰的时候要吃梨，清明的时候要扫墓、踏青。

**小知识**：
"春雨惊春清谷天"，这句节气歌里就包含了春天的六个节气，你记住了吗？

### 传统文化：清明习俗
清明节是我们国家的传统节日，也是二十四节气之一。清明节的习俗有很多：
1. **扫墓祭祖**：纪念去世的亲人，表达思念之情
2. **踏青**：春天到了，到郊外去游玩，感受春天的美好
3. **插柳**：折几根柳枝插在门上，可以避邪
4. **放风筝**：把风筝放上蓝天，寓意放走晦气，带来好运
5. **吃青团**：用艾草和糯米粉做的青团，是清明节的传统美食

### 古诗词里的春天
古代的诗人写了很多赞美春天的优美诗句：
- "等闲识得东风面，万紫千红总是春。"（朱熹《春日》）
- "春眠不觉晓，处处闻啼鸟。"（孟浩然《春晓》）
- "好雨知时节，当春乃发生。"（杜甫《春夜喜雨》）
- "竹外桃花三两枝，春江水暖鸭先知。"（苏轼《惠崇春江晚景》）

这些诗句生动地描绘了春天的美丽景色，千百年来一直被人们传诵。

## 单元评价建议
1. **朗读背诵评价**：检查学生的课文朗读和古诗背诵情况，要求读音准确，有感情
2. **观察表达评价**：评价学生观察春天、描述春天的表现，要求观察仔细，表达清楚
3. **实践活动评价**：评价学生完成拓展任务的情况，鼓励学生多接触大自然，多实践
4. **综合评价**：结合学生的学习和实践表现，颁发"观察小能手"、"环保小卫士"、"小诗人"等奖状，培养学生对大自然的热爱之情
