# 一年级下册 第三单元 奇妙的大自然

## 单元主题
探索大自然的奥秘，培养好奇心和探索精神，了解自然现象背后的科学道理，学会尊重自然、保护自然。

## 语文要素
1. 认识"星、云、雷、电、风、雨"等35个常用汉字
2. 会写"天、地、日、月、星、云"等12个生字
3. 能正确、流利、有感情地朗读课文，背诵关于自然现象的儿歌和古诗
4. 能观察简单的自然现象，用几句话描述自己的发现
5. 学习积累描写自然现象的词语和句子

## 文化要素
1. 了解中国古代关于自然现象的神话传说
2. 学习古人观察自然、探索自然的智慧
3. 了解二十四节气中的"立夏"、"小满"、"芒种"三个夏季节气

## 一、课文板块

### 课文1：彩虹
雨停了，天上有一座美丽的桥。
爸爸，如果我提着你那把浇花用的水壶，走到桥上去，把水洒下来，不是我在下雨吗？我把雨洒在山上的田地里，你就不用挑水去浇了，你高兴吗？
妈妈，如果我拿着你梳头用的那面圆圆的镜子，走到桥上去，天上不就多了一个月亮吗？我拿着圆圆的月亮照着你梳头，你高兴吗？
哥哥，如果我把你系在门前树上的秋千，拿去挂在彩虹桥上，坐着秋千荡来荡去，花裙子飘啊飘的，不就成了一朵彩云吗？你看见了，高兴吗？

**生字表**：
会认：虹、座、提、浇、洒、挑、兴、镜、拿、照、千、裙
会写：那、着、到、高、兴、千、成、为、天、空

**课文讲解**：
雨后的彩虹像一座美丽的桥，小作者发挥丰富的想象力，想在彩虹桥上为家人做很多事情。大自然中有很多奇妙的现象，等待我们去发现和探索。

### 课文2：要下雨了
小白兔弯着腰在山坡上割草。天阴沉沉的，小白兔直起身子，伸了伸腰。
小燕子从他头上飞过。小白兔大声喊："燕子，燕子，你为什么飞得这么低呀？"
燕子边飞边说："要下雨了，空气很潮湿，虫子的翅膀沾了小水珠，飞不高。我正忙着捉虫子呢！"
是要下雨了吗？小白兔往前边池子里一看，小鱼都游到水面上来了。
小白兔跑过去，问："小鱼，小鱼，今天怎么有空出来呀？"
小鱼说："要下雨了，水里闷得很，我们到水面上来透透气。小白兔，你快回家吧，小心淋着雨。"
小白兔连忙挎起篮子往家跑。他看见路边有一大群蚂蚁，就把要下雨的消息告诉了蚂蚁。一只大蚂蚁说："是要下雨了，我们正忙着搬东西呢！"
小白兔加快步子往家跑。他一边跑一边喊："妈妈，妈妈，要下雨了！"
轰隆隆，天空响起了一阵雷声。哗，哗，哗，大雨真的下起来了！

**生字表**：
会认：腰、坡、沉、伸、潮、湿、呢、空、闷、消、息、搬、响
会写：吗、吧、呀、呢、边、加、直、呀、边、呢

**课文讲解**：
下雨之前有很多预兆：燕子低飞、小鱼游到水面、蚂蚁搬家。只要我们仔细观察，就会发现大自然中有很多小秘密。

### 课文3：小池
（宋）杨万里
泉眼无声惜细流，
树阴照水爱晴柔。
小荷才露尖尖角，
早有蜻蜓立上头。

**生字表**：
会认：泉、眼、惜、流、爱、柔、荷、露、角、蜻、蜓
会写：首、采、无、树、爱、尖、角、亮、机、台

**课文讲解**：
这首诗描写了夏天小池塘的美丽景色：泉水静静地流着，树荫倒映在水面上，小小的荷叶刚刚长出尖尖的角，就有蜻蜓停在上面。大自然的景色真美啊！

荷叶圆圆的，绿绿的。

## 二、阅读材料

### 阅读1：大自然的语言
别以为人才会说话，
大自然也有语言。
这语言到处都有，
仔细观察就能发现。
白云飘得高高，
明天准是晴天。
蚂蚁往高处搬家，
出门要带雨伞。
树桩上有一道道圈圈，
一圈就是一年。
大自然的语言啊，
真是妙不可言。
不爱学习的人看不懂，
粗心大意的人永远看不见。

**阅读提示**：
大自然就像一本厚厚的书，里面藏着很多知识。只要我们仔细观察，认真思考，就能读懂大自然的语言。

---

### 阅读2：盘古开天地
很久很久以前，天和地还没有分开，宇宙混沌一片。有个叫盘古的巨人，在这混沌之中，一直睡了一万八千年。
有一天，盘古突然醒了。他见周围一片漆黑，就抡起大斧头，朝眼前的黑暗猛劈过去。只听一声巨响，混沌一片的东西渐渐分开了。轻而清的东西，缓缓上升，变成了天；重而浊的东西，慢慢下降，变成了地。
天和地分开以后，盘古怕它们还会合在一起，就头顶着天，用脚使劲蹬着地。天每天升高一丈，地每天加厚一丈，盘古的身体也跟着长高。这样过了一万八千年，天升得高极了，地变得厚极了。盘古也累得倒了下去。
盘古倒下以后，他的身体发生了巨大的变化。他呼出的气息变成了四季的风和飘动的云；他发出的声音化作了隆隆的雷声；他的左眼变成了太阳，照耀大地；他的右眼变成了月亮，给夜晚带来光明；他的四肢和躯干变成了大地的四极和五方的名山；他的血液变成了奔流不息的江河；他的汗毛变成了茂盛的花草树木；他的汗水变成了滋润万物的雨露……
伟大的巨人盘古，用他的整个身体创造了美丽的宇宙。

**阅读提示**：
这是中国古代的神话传说，古人用这样的故事来解释天地万物的由来。现在我们已经知道了宇宙和地球的真正起源，但是这个美丽的神话故事还是一直流传到今天。

### 阅读3：为什么天空是蓝色的？
小朋友们，你们有没有想过，为什么天空是蓝色的呢？
原来，太阳光有七种颜色：红、橙、黄、绿、蓝、靛、紫。其中红色、橙色、黄色的光波长比较长，能够直接穿过大气层射到地面上；而蓝色、紫色的光波长比较短，很容易被空气中的灰尘、水滴散射，所以天空看起来就是蓝色的。
大自然是不是很奇妙呀！

## 三、练习活动

### 活动1：自然现象小问号
**目标**：培养好奇心和探索精神
**过程**：
1. 让学生提出自己对大自然的疑问，比如"为什么会下雨？""为什么星星会眨眼睛？""为什么月亮有时候圆有时候弯？"
2. 把大家的问题写在黑板上
3. 能解答的问题大家一起讨论解答，解答不了的问题让学生回家和爸爸妈妈一起查资料
4. 下节课分享自己找到的答案

### 活动2：天气谚语大比拼
**目标**：积累民间智慧，了解古人观察自然的经验
**过程**：
1. 老师先介绍一些天气谚语，比如"朝霞不出门，晚霞行千里"、"燕子低飞蛇过道，大雨不久就来到"
2. 让学生说说自己知道的其他天气谚语
3. 讨论这些谚语有没有道理，为什么
4. 把大家收集的谚语整理出来，做成一张天气谚语小报

### 活动3：我的自然观察日记
**目标**：培养观察能力和写作能力
**过程**：
1. 选择一种自然现象或者一种小动物、植物，连续观察一周
2. 每天记录下自己的观察发现
3. 一周后，和同学们分享自己的观察日记
4. 评选出"最佳观察日记"

## 四、拓展任务

### 任务1：我是小小气象员
**跨学科融合**：语文 + 科学 + 生活实践
**要求**：
1. 连续一周观察每天的天气情况，记录下气温、风向、天气状况（晴天、阴天、雨天）
2. 学习看天气预报，了解天气符号
3. 每天早上当"气象员"，给同学们播报当天的天气
4. 一周后，总结这周的天气特点，看看有没有什么规律

### 任务2：制作自然标本
**跨学科融合**：语文 + 美术 + 科学
**要求**：
1. 和爸爸妈妈一起到郊外去，收集一些漂亮的树叶、花瓣、小石子
2. 把这些东西压平、晾干，做成标本
3. 给每个标本贴上标签，写上名称、采集时间、采集地点
4. 把你的标本带到学校，和同学们分享，介绍你收集的标本

### 任务3：星空探秘
**跨学科融合**：语文 + 科学 + 天文
**要求**：
1. 在爸爸妈妈的陪伴下，在天气晴朗的晚上观察星空
2. 找找看，你能认出哪些星星？有没有找到北斗七星？
3. 问问爸爸妈妈，或者查资料，了解一些关于星星的知识
4. 画一幅你看到的星空，写几句话说说你观察星空的感受

## 五、文化延伸

### 传统文化：二十四节气之夏天的节气
夏天有六个节气，我们先学习前三个：
1. **立夏**：夏天的开始，天气开始变热
2. **小满**：麦类等夏熟作物的籽粒开始饱满，但还没有成熟
3. **芒种**：麦类等有芒作物成熟，是一年中最忙的夏收夏种时节
4. **夏至**：这一天白天最长，夜晚最短
5. **小暑**：天气开始炎热，但还不是最热的时候
6. **大暑**：一年中最热的时候

每个节气都有不同的习俗，比如立夏要吃立夏蛋，芒种要送花神。

### 传统文化：古人的自然智慧
我们中国古代的劳动人民非常聪明，他们通过长期观察自然，总结出了很多经验：
- 他们根据二十四节气来安排农业生产，什么时候播种，什么时候收割，都有严格的时间
- 他们总结出了很多天气谚语，能够预测天气的变化
- 他们发明了指南针、造纸术、印刷术、火药四大发明，为人类文明做出了巨大贡献

这些都是古人智慧的结晶，值得我们学习和继承。

### 传统神话：自然传说
中国古代有很多关于自然现象的美丽神话：
- **女娲补天**：传说天破了一个大洞，女娲炼了五色石把天补好
- **后羿射日**：传说天上有十个太阳，晒得庄稼枯死，后羿射下了九个太阳，救了老百姓
- **嫦娥奔月**：传说嫦娥吃了不死药，飞到了月亮上，住在月宫里
- **精卫填海**：传说炎帝的女儿淹死在东海里，变成了精卫鸟，每天衔来石头和树枝，想要把东海填平

这些神话故事反映了古人探索自然、征服自然的美好愿望。

## 单元评价建议
1. **朗读背诵评价**：检查学生的课文朗读和古诗背诵情况，要求读音准确，有感情
2. **观察表达评价**：评价学生观察自然、描述自然现象的表现，要求观察仔细，表达清楚
3. **实践活动评价**：评价学生完成拓展任务的情况，鼓励学生多观察、多探索、多思考
4. **综合评价**：结合学生的学习和实践表现，颁发"小小科学家"、"观察小能手"、"自然小博士"等奖状，培养学生的好奇心和探索精神
