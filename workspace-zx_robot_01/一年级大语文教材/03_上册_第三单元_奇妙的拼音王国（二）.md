# 一年级上册 第三单元 奇妙的拼音王国（二）

## 单元主题
继续学习剩余的声母、复韵母、前鼻韵母和后鼻韵母，掌握三拼音节和整体认读音节，能够熟练拼读汉字，为独立阅读打下坚实基础。

## 语文要素
1. 认识剩余的13个声母：j、q、x、zh、ch、sh、r、z、c、s、y、w
2. 认识9个复韵母：ai、ei、ui、ao、ou、iu、ie、üe、er
3. 认识5个前鼻韵母：an、en、in、un、ün
4. 认识4个后鼻韵母：ang、eng、ing、ong
5. 掌握16个整体认读音节
6. 学会三拼音节的拼读方法
7. 会认20个常用汉字，会写10个生字

## 文化要素
1. 了解拼音在汉字信息化中的重要作用
2. 体会拼音作为文化传播工具的价值

## 一、课文板块

### 课文1：声母大家族集合啦
今天，声母家族的其他小伙伴也来和我们见面了，它们是：
j、q、x、zh、ch、sh、r、z、c、s、y、w

**拼音学习**：
- **j**：公鸡鸣叫j j j，竖弯加点j j j
- **q**：气球飞起q q q，左上半圆q q q
- **x**：西瓜刀切x x x，像个叉叉x x x
- **zh**：蜘蛛织网zh zh zh，z加椅子zh zh zh
- **ch**：小猫吃鱼ch ch ch，c加椅子ch ch ch
- **sh**：狮子吃草sh sh sh，s加椅子sh sh sh
- **r**：小苗发芽r r r，日头高照r r r
- **z**：小羊写字z z z，像个2字z z z
- **c**：刺猬缩团c c c，半个圆圈c c c
- **s**：春蚕吐丝s s s，半个8字s s s
- **y**：树杈挂衣y y y，像个树杈y y y
- **w**：屋顶相连w w w，波浪起伏w w w

**小规则**：
j、q、x真淘气，
从不和ü在一起，
它们和ü来相拼，
见了鱼眼就挖去。
（小ü小ü有礼貌，见了j q x就脱帽）

### 课文2：复韵母和鼻韵母
韵母家族还有更多的成员，它们是复韵母和鼻韵母：
复韵母：ai、ei、ui、ao、ou、iu、ie、üe、er
前鼻韵母：an、en、in、un、ün
后鼻韵母：ang、eng、ing、ong

**拼音学习**：
- **ai**：高矮的ai，a在前i在后，ai ai ai
- **ei**：杯子的ei，e在前i在后，ei ei ei
- **ui**：围巾的ui，u在前i在后，ui ui ui
- **ao**：棉袄的ao，a在前o在后，ao ao ao
- **ou**：海鸥的ou，o在前u在后，ou ou ou
- **iu**：邮票的iu，i在前u在后，iu iu iu
- **ie**：椰子的ie，i在前e在后，ie ie ie
- **üe**：月亮的üe，ü在前e在后，üe üe üe
- **er**：耳朵的er，er er er，是个特殊韵母，不和声母拼
- **an**：天安门的an，an an an
- **en**：摁门铃的en，en en en
- **in**：阴天的in，in in in
- **un**：蚊子的un，un un un
- **ün**：白云的ün，ün ün ün
- **ang**：昂首的ang，ang ang ang
- **eng**：台灯的eng，eng eng eng
- **ing**：老鹰的ing，ing ing ing
- **ong**：钟声的ong，ong ong ong

### 课文3：整体认读音节
拼音王国里还有16个特殊的音节，它们不用拼读，直接整体认读：
zhi、chi、shi、ri、zi、ci、si、
yi、wu、yu、ye、yue、yuan、
yin、yun、ying

**记忆儿歌**：
zh ch sh r z c s，
自成音节后加i。
yi wu yu 一无雨，
ye yue yuan 夜月圆，
yin yun ying 隐云影。

**三拼音节拼读方法**：
声轻介快韵母响，
三音连读很顺当。
比如：g-u-a → gua（西瓜的瓜）
q-i-ao → qiao（小桥的桥）

**生字表**：
会认：瓜、果、桃、李、桥、家、飞、机、河、湖、
      山、石、田、土、红、黄、蓝、绿、白、黑
会写：日、月、水、火、山、石、田、土、禾、木

### 阅读2：汉语拼音的故事
你知道吗？汉语拼音的发明可不容易呢！
在100多年前，就有很多学者开始研究如何给汉字注音。
新中国成立后，国家组织了很多专家，经过多年的研究，终于在1958年正式公布了《汉语拼音方案》。
现在，汉语拼音不仅是我们学习汉字的工具，还成为了国际标准，全世界的人都可以通过拼音来学习汉语，了解中国文化。

**阅读提示**：

## 三、练习活动

### 活动1：拼音超市大购物
**目标**：练习拼读，提高拼音运用能力
**准备**：在教室里布置"拼音超市"，各种物品上都贴上拼音标签
**过程**：
1. 每个学生拿到10元"拼音币"
2. 学生可以自由"购买"物品，但必须准确读出物品上的拼音才能购买
3. 老师当收银员，检查学生的拼读是否正确
4. 购物结束后，比比谁买的东西最多，拼读最准确

### 活动2：拼音接龙讲故事
**目标**：练习拼音拼读，培养想象力和表达能力
**过程**：
1. 老师给出一个拼音词，比如"xiǎo bái tù（小白兔）"
2. 第一个学生用这个词说一句话，然后再出一个新的拼音词
3. 第二个学生用上一个同学出的词继续讲故事
4. 这样依次接龙，看看最后会编成一个什么样的有趣故事
5. 表扬编故事最有创意的同学

### 活动3：拼音查字典比赛
**目标**：学会用音序查字法查字典
**准备**：每个学生一本新华字典
**过程**：
1. 老师说一个字，比如"花"
2. 学生快速用音序查字法在字典里找到这个字
3. 谁先找到，并且读准字音，说出意思，谁就获胜
4. 多进行几轮，评选出"查字典小能手"

## 四、拓展任务

### 任务1：我的拼音绘本
**跨学科融合**：语文 + 美术 + 创意写作
**要求**：
1. 用我们学过的拼音写一个小故事
2. 给故事配上漂亮的图画
3. 故事不用太长，3-5句话就可以
4. 可以是关于小动物的，也可以是关于你的生活的
5. 把你的拼音绘本读给爸爸妈妈听

### 任务2：拼音广告牌
**跨学科融合**：语文 + 生活实践
**要求**：
1. 和爸爸妈妈一起上街的时候，看看路边的广告牌、商店招牌
2. 找找上面有没有你认识的拼音
3. 试着拼一拼，看看是什么字
4. 把你找到的拼音记录下来，和同学们分享

### 任务3：拼音打字小能手
**跨学科融合**：语文 + 信息技术
**要求**：
1. 在爸爸妈妈的帮助下，打开电脑的拼音输入法
2. 试着用拼音打几个你会写的字，比如你的名字、爸爸妈妈的名字
3. 看看你一分钟能打多少个字
4. 学会了拼音打字，以后你就可以用电脑写日记啦

## 五、文化延伸

### 传统文化：汉字的魅力
小朋友们，我们的汉字是世界上最古老的文字之一，已经有几千年的历史了。
每个汉字都有它的意思，有的汉字还像一幅画呢！比如：
- "山"字就像一座高高的山
- "水"字就像流动的河水
- "日"字就像圆圆的太阳
- "月"字就像弯弯的月亮

拼音就像一把钥匙，帮我们打开汉字王国的大门，让我们认识更多的汉字，了解更多的中国文化。

**小知识**：
世界上现在还有很多人在学习汉字和汉语呢！我们作为中国人，更要学好我们的母语，把中国文化传承下去。

### 拼音与科技
拼音不仅帮助我们学习汉字，在科技方面的用处也很大：
1. 我们用的手机、电脑，大部分都是用拼音输入法打字的
2. 很多电子产品的操作界面都有拼音提示
3. 人工智能语音识别也离不开拼音
4. 我们的北斗卫星导航系统里的地名，也是用拼音标注的

**小讨论**：
小朋友们，你们想一想，如果没有拼音，我们的生活会有什么不方便呢？和同学们说一说。
