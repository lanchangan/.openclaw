# 一年级大语文教材 学生练习册

## 使用说明
这本练习册是和《一年级大语文教材》配套使用的，分为上册和下册两部分，每单元都有基础练习、提高练习和拓展练习三个部分，同学们可以根据自己的情况选择使用。

做题的时候要认真读题，仔细思考，字迹工整，做完了要认真检查哦！

## 上册 第一单元 我是小学生啦

### 基础练习
1. 读一读，连一连
   学 校 老 师 同 学 书 包
   lǎo shī tóng xué shū bāo xué xiào

2. 我会写
   上（ ） 下（ ） 大（ ） 小（ ） 人（ ）

3. 我会组词
   学（ ） 校（ ） 老（ ） 师（ ） 同（ ）

### 提高练习
1. 填空
   我们的学校有______，有______，还有______。
   我喜欢我们的学校，因为______。

2. 背一背《上学歌》，并试着写一写（不会写的字用拼音代替）
   太阳当空照，____________。
   我去上学校，____________。

3. 说一说
   你上学第一天的心情是什么样的？和爸爸妈妈说一说。

### 拓展练习
1. 画一画你最喜欢的校园一角，并用一两句话介绍一下。
2. 向爸爸妈妈介绍一下你认识的新朋友。
3. 做一张好习惯打卡表，每天坚持做到：按时上学、自己整理书包、认真听讲。

## 上册 第二单元 奇妙的拼音王国（一）

### 基础练习
1. 读一读，写一写
   单韵母：a o e i u ü
   声母：b p m f d t n l g k h

2. 我会拼
   b - a → ( ) p - o → ( ) m - i → ( )
   f - u → ( ) d - e → ( ) t - u → ( )

3. 我会标声调
   ba（一声）( ) po（二声）( ) mi（三声）( )
   fu（四声）( ) de（轻声）( ) tu（二声）( )

### 提高练习
1. 找朋友，把能组成音节的连起来
   b p m f d t
   a o e i u ü

2. 读一读，比一比，看看有什么不同
   b - d p - q f - t n - l

3. 用拼音写一写你会的物品名称，至少写3个。

### 拓展练习
1. 在家里找一找，哪些物品的名称里有我们学过的音节，记录下来。
2. 做拼音卡片，和爸爸妈妈一起玩拼音接龙游戏。
3. 把学过的拼音字母画成有趣的图画。

## 上册 第三单元 奇妙的拼音王国（二）

### 基础练习
1. 读一读，写一写
   声母：j q x zh ch sh r z c s y w
   复韵母：ai ei ui ao ou iu ie üe er
   前鼻韵母：an en in un ün
   后鼻韵母：ang eng ing ong
   整体认读音节：zhi chi shi ri zi ci si yi wu yu ye yue yuan yin yun ying

2. 我会拼
   j - i - a → ( ) q - ü → ( ) x - üe → ( )
   zh - u - ang → ( ) ch - eng → ( ) sh - ui → ( )

3. 我会读拼音写词语
   dà mǐ ( ) tǔ dì ( ) mǎ ěr ( )
   huā duǒ ( ) gē ge ( ) dì di ( )

### 提高练习
1. 给下面的音节分类
   ri guo zhi si jia qiao yuan yin ying
   三拼音节：________________________
   整体认读音节：____________________

2. 用拼音写一句话，介绍一下你的家。
3. 查字典比赛：用音序查字法查"花"、"草"、"山"、"水"四个字，看看谁查得快。

### 拓展练习
1. 用拼音写一个小故事，配上图画，做成一本拼音绘本。
2. 上街的时候注意观察路边的广告牌，看看有没有你认识的拼音，记录下来。
3. 在爸爸妈妈的帮助下，试着用拼音输入法打几个字。

## 上册 第四单元 美丽的大自然

### 基础练习
1. 读一读，连一连
   秋 天 树 叶 星 星 江 南
   shù yè qiū tiān jiāng nán xīng xing

2. 我会写
   春（ ） 夏（ ） 秋（ ） 冬（ ） 花（ ）
   草（ ） 鱼（ ） 鸟（ ） 东（ ） 西（ ）

3. 我会填
   弯弯的（ ） 闪闪的（ ） 蓝蓝的（ ）
   圆圆的（ ） 尖尖的（ ） 弯弯的（ ）

### 提高练习
1. 背一背《江南》和《小小的船》，并默写。
2. 填空
   秋天来了，天气______，树叶______，大雁______。
   我喜欢秋天，因为______。

3. 说一说
   你最喜欢哪个季节？为什么？

### 拓展练习
1. 和爸爸妈妈一起去寻找秋天，收集不同的落叶，做一幅落叶贴画。
2. 连续一周观察天气，做一个天气记录表。
3. 学着写一首关于季节的小诗。

## 上册 第七单元 四季的故事

### 基础练习
1. 读一读，写一写
   春 风 花 草 树 木 雪 雨 雷 电

2. 我会填四季的词语
   春：______ ______ ______
   夏：______ ______ ______
   秋：______ ______ ______
   冬：______ ______ ______

3. 背一背《江雪》，并默写。

### 提高练习
1. 想一想，连一连
   春天 热，人们穿短袖
   夏天 凉，树叶黄了
   秋天 冷，会下雪
   冬天 暖，小草发芽

2. 写几句话，说说你最喜欢的季节是什么样子的。
3. 二十四节气歌里的"春雨惊春清谷天"包含了哪六个节气？

### 拓展练习
1. 选择一个季节，连续观察一周，写一篇观察日记。
2. 和爸爸妈妈一起做一道和季节相关的美食，记录下做法和味道。
3. 选择一首关于四季的古诗，给它配上漂亮的图画。

## 上册 第八单元 我们的节日

3. 过年的时候给长辈拜年，说几句祝福的话。
