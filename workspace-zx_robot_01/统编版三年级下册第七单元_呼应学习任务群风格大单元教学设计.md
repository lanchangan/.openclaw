# 统编版小学语文三年级下册第七单元
## 大单元整体教学设计（呼应学习任务群风格）

---

## 一、单元整体规划

### 单元基本信息
- **单元主题**：奇妙的世界
- **教材篇目**：《我们奇妙的世界》《暴风雨来临之前》《火烧云》
- **语文园地**+**习作**：我的奇思妙想
- **课时安排**：共计11课时

### 单元大概念（语文要素）
**"发现世界的奇妙，学习按一定顺序写景物，感受大自然的美"**

### 单元核心目标
| 维度 | 目标内容 |
|------|----------|
| **认知目标** | 1. 认识本单元生字词语，正确书写<br>2. 能正确、流利、有感情地朗读课文<br>3. 了解作者是怎样发现奇妙、描写奇妙的 |
| **能力目标** | 1. 学习作者按照一定顺序观察和描写景物的方法<br>2. 能发现生活中的奇妙事物，按照一定顺序写下来<br>3. 初步培养学生的观察能力和想象能力 |
| **情感目标** | 1. 感受世界的奇妙，培养热爱大自然的情感<br>2. 养成留心观察周围事物的好习惯<br>3. 激发学生探索世界奥秘的兴趣 |

### 单元核心任务
**"制作一本《奇妙世界观察手册》——记录你发现的奇妙"**

> 设计理念：呼应"语文-生活-语文"，从课文学习发现描写奇妙的方法，到生活中寻找奇妙，再回到语文中表达奇妙。

---

## 二、单元学习进程安排

| 学习阶段 | 课时 | 学习内容 |
|----------|------|----------|
| **第一阶段：单元起航——发现奇妙** | 1课时 | 单元导入，激发兴趣，明确任务 |
| **第二阶段：精读感悟——品味奇妙** | 6课时 | 《我们奇妙的世界》2课时<br>《暴风雨来临之前》2课时<br>《火烧云》2课时 |
| **第三阶段：交流探究——寻找奇妙** | 2课时 | 交流平台+初试身手+观察实践 |
| **第四阶段：表达展示——记录奇妙** | 2课时 | 习作《我的奇思妙想》+单元总结展示 |
| **总计** | **11课时** | |

## 三、教学设计逐课时展开

---

### 第一阶段：单元起航——"开启奇妙发现之旅"
**课时：1课时**

#### 学习目标：
1. 感受单元主题，产生阅读期待
2. 了解单元学习任务，明确学习要求
3. 初步交流"我发现的奇妙"

#### 教学过程：

##### （一）游戏导入："找找看，哪里奇妙？"（5分钟）
- 教师出示几张奇妙自然图片（极光、彩虹、火烧云、雾凇等）
- 学生抢答："这是什么？它哪里奇妙？"
- 教师小结：我们生活的世界本身就是一个奇妙的世界

##### （二）单元导读："这单元我们学什么？"（10分钟）
- 学生自由读单元导语
  > "天空向我们展示了许多秘密。……只要我们仔细地观察、寻找，就能从极普通的事物中找到美。"
- 师生谈话：
  - 什么是"奇妙"？你眼中的奇妙是什么？
  - 这个单元我们要学习什么？
- 提炼单元语文要素：**发现奇妙→学习写法→记录奇妙**

##### （三）任务揭晓："我们一起来做《奇妙世界观察手册》"（15分钟）
- 教师展示样例手册
- 说明手册内容：
  1. **课文奇妙摘录**：从三篇课文中摘抄你认为最奇妙的句子
  2. **我的奇妙发现**：生活中我发现的奇妙事物
  3. **我的奇思妙想**：写写自己的奇妙想法
  4. **奇妙美图欣赏**：贴一贴画一画你发现的奇妙
- 学生领取空白手册，写下自己的第一则发现："我现在发现______真奇妙，因为______。"

##### （四）布置预习："先睹为快读课文"（5分钟）
- 学生自由朗读三篇课文
- 标出自然段，圈出生字新词
- 想一想：三篇课文分别写了什么奇妙？

**设计呼应**：遵循"总起-分述-总结"，开门见山揭示单元主题，用核心任务驱动整个单元学习。

---

### 第二阶段：精读感悟——品味奇妙

---

#### 《我们奇妙的世界》 第1课时："天地间的奇妙宝藏"
**学习目标**：
1. 认识生字词语，正确书写
2. 正确朗读课文，理清课文叙述顺序
3. 感受天空的奇妙，学习作者按照时间顺序描写的方法

**教学过程**：

##### 1. 谈话导入（5分钟）
- 上节课我们启动了奇妙发现之旅，谁来说说你预习了课文，发现这一课写了什么奇妙？
- 板书课题：我们奇妙的世界

##### 2. 初读感知（10分钟）
- 学生自由读课文，要求：
  - 读准字音，读通句子
  - 想一想：课文是从哪几个方面写世界的奇妙？
- 检查生字认读
- 交流发现：课文写了**天空**和**大地**两个方面的奇妙

##### 3. 品味天空的奇妙（20分钟）
- 学生默读课文第2-8自然段
- 思考：作者是按照什么顺序写天空的奇妙？
  - 日出→云彩→雨后→黄昏→夜晚（**时间顺序**）
- 小组合作填写"天空奇妙时间表"：

| 时间 | 奇妙景象 | 我的感受 |
|------|----------|----------|
| 清晨 |  |  |
| 晴天 |  |  |
| 雨后 |  |  |
| 黄昏 |  |  |
| 夜晚 |  |  |

- 分享交流：你最喜欢哪个时间段的奇妙？为什么？
- 指导朗读：读出惊叹、赞美之情

##### 4. 小结写法+写字（5分钟）
- 小结：作者按照时间顺序，把一天中天空的奇妙写得清清楚楚
- 指导书写生字：呈、幻、蜡、烛……

**呼应设计**：遵循"发现（顺序）-感受（奇妙）-分享（交流）-运用（填写表格）"四环节。

---

#### 《我们奇妙的世界》 第2课时："大地上的奇妙发现"
**学习目标**：
1. 继续学习课文，感受大地四季的奇妙
2. 体会作者"仔细观察，寻找美"的思想
3. 练习仿写，学习从普通事物中发现美

**教学过程**：

##### 1. 复习导入（5分钟）
- 上节课我们欣赏了天空的奇妙，作者按什么顺序写的？
- 这节课我们来看大地上的奇妙

##### 2. 品读大地四季的奇妙（15分钟）
- 学生默读第9-16自然段
- 思考：作者又是按照什么顺序写大地的？（**季节顺序**）
- 四人小组合作：
  - 每个同学负责一个季节
  - 读一读，说说这个季节哪里奇妙
- 全班交流：
  - 春天：种子生根发芽，世界一切都在改变
  - 夏天：大树带来绿荫，水滴闪闪发光
  - 秋天：金黄的花朵，果实闪耀着光芒
  -  冬天：冰柱锋利，水滴像珍珠

##### 3. 领悟写法："极普通的事物也有美"（10分钟）
- 齐读最后一个自然段：
  > "是的，世界上存在的奇妙的事物是无穷的，只要我们去寻找。"
- 讨论交流：
  - 课文中写了哪些"极普通的事物"？（鹅卵石、冰块、羽毛……）
  - 你从哪些普通事物中发现过美？
- 完成课后练习：读一读，你也来写一句

  > 你看，______就像______。

##### 4. 迁移练习："我的奇妙发现"（8分钟）
- 在你的《奇妙世界观察手册》上，写下一个你在普通事物中发现的奇妙
- 同桌互相交流
- 指导书写剩余生字

##### 5. 小结（2分钟）
- 奇妙无处不在，只要我们仔细观察、寻找

**呼应设计**：继续贯彻"发现-感受-分享-运用"，落实"从普通事物中发现美"这一关键感悟。

---

#### 《暴风雨来临之前》 第1课时："暴风雨前的奇妙景象"
**学习目标**：
1. 认识生字词语，正确书写
2. 正确朗读课文，理清课文顺序
3. 感受暴风雨来临前天空和动物的奇妙变化

**教学过程**：

##### 1. 激趣导入（5分钟）
- 谈话：你经历过暴风雨吗？暴风雨来临之前，天气有什么变化？
- 板书课题：暴风雨来临之前

##### 2. 初读课文，整体感知（10分钟）
- 学生自由朗读课文，读准字音
- 想一想：课文是按照什么顺序写的？
  - **暴风雨来临之前**→**暴风雨之中**→**暴风雨之后**
- 这节课我们先学习"暴风雨来临之前"

##### 3. 品读感悟："天空的变化"（15分钟）
- 默读课文描写暴风雨来临之前的部分
- 思考：
  - 作者写了哪些事物的变化？（天空、动物）
  - 填写"暴风雨来临前变化时间表"：

| 时间推移 | 天空的变化 | 动物的活动 |
|----------|------------|------------|
| 一开始 |  |  |
| 过了一会儿 |  |  |
| 不久 |  |  |

- 交流分享：
  - 乌云是怎样变化的？（从"一块白"到"乌云滚滚"）
  - 动物有什么反应？（蝉不叫了→蜘蛛收网→蜻蜓低飞）
  - 作者的心情有什么变化？（从惬意→好奇→闷热）

##### 4. 小结写字（5分钟）
- 作者按照**时间推移顺序**写出了暴风雨来临前的变化
- 指导书写生字：咆、哮、僵、坠……

**呼应设计**：用"变化时间表"梳理顺序，符合呼应一书"抓住顺序、理清脉络"的教学设计特色。

---

#### 《暴风雨来临之前》 第2课时："风雨过后更清新"
**学习目标**：
1. 继续学习课文，感受暴风雨中和雨后的奇妙
2. 体会作者对大自然的热爱之情
3. 学习作者细致观察、生动描写的方法

**教学过程**：

##### 1. 复习导入（5分钟）
- 上节课我们看到暴风雨来临前有哪些奇妙变化？
- 这节课我们来看暴风雨到来和过后的景象

##### 2. 品读暴风雨之中（10分钟）
- 指名朗读
- 说一说：暴风雨到来时，你感受到什么？（狂风、雷电、大雨，非常壮观）
- 圈画出你认为写得好的词语：狂风咆哮、闪电划破、雷声轰鸣……

##### 3. 品味雨后的奇妙（15分钟）
- 学生自由读雨后部分
- 小组讨论：
  - 雨后的世界和之前有什么不一样？
  - 作者写了哪些景物？（天空→彩虹→树叶→蚂蚁→蜗牛）
  - 为什么说"一场暴风雨过后，一切都变了"？
- 美读分享：选择你认为最奇妙的句子读给大家听
  > "朋友，暴风雨过后一切都变了，你发现了吗？"

##### 4. 感悟写法+迁移练习（5分钟）
- 讨论：整篇课文按照什么顺序写的？（**时间顺序：前-中-后**）
- 在《奇妙世界观察手册》中摘抄你最喜欢的描写雨后奇妙的句子
- 指导书写剩余生字

**呼应设计**：延续"时间线"写法学习，前后呼应，强化"按顺序写"这一语文要素。

---

#### 《火烧云》 第1课时："晚饭过后，火烧云上来了"
**学习目标**：
1. 认识生字词语，正确书写
2. 正确流利朗读课文，感受火烧云的奇妙
3. 学习作者描写火烧云颜色变化的方法

**教学过程**：

##### 1. 图片导入（5分钟）
- 教师出示火烧云图片
- 谈话：你见过火烧云吗？它是什么样子的？
- 板书课题：火烧云

##### 2. 初读感知（10分钟）
- 学生自由读课文
- 想一想：课文是按照什么顺序写火烧云的？
  - **上来了→变化着→下去了**
- 这节课我们学习火烧云上来了，颜色在变化

##### 3. 品读火烧云的颜色变化（20分钟）
- 默读课文，思考：
  - 火烧云颜色有什么特点？（**变化极多**）
  - 作者写了哪些颜色？
- 学生整理：
  - 红彤彤、金灿灿、半紫半黄、半灰半百合色……
  - 还有很多说不出来、见也没见过的颜色
- 完成"火烧云颜色变化卡"：

  我找到的颜色：
  1. _________
  2. _________
  3. _________

  我还能想象：
  - _____、_____、_____

- 朗读比赛：读出火烧云的绚丽多彩

##### 4. 小结写字（5分钟）
- 火烧云颜色真奇妙，变化多端真好看
- 指导书写：彤、檀、盈、凶……

**呼应设计**：用"找一找、填一填"落实"发现变化"，体现呼应一书"语文实践"特色。

---

#### 《火烧云》 第2课时："火烧云形状奇妙多"
**学习目标**：
1. 感受火烧云形状的奇妙变化
2. 学习作者描写形状变化的方法
3. 积累优美语句，练习想象表达

**教学过程**：

##### 1. 复习导入（5分钟）
- 上节课我们感受了火烧云颜色的奇妙，有什么特点？
- 这节课来看火烧云形状的奇妙

##### 2. 品读形状变化（15分钟）
- 学生默读课文
- 思考：作者写了火烧云哪些形状？
  - 一匹马→大狗→狮子（**变化极快**）
- 小组合作完成"火烧云形状变化时间表"：

| 形状 | 出现 | 变化 | 消失 |
|------|------|------|------|
| 一匹马 |  |  |  |
| 一条大狗 |  |  |  |
| 一头大狮子 |  |  |  |

- 交流分享：
  - 形状变化有什么特点？（出现快，变化快，消失快）
  - 从哪些词语可以看出来？（一会儿、忽然、接着、一转眼……）

##### 3. 感悟写法+想象练习（15分钟）
- 讨论：作者为什么能把火烧云写得这么生动有趣？
  - 细致观察+丰富想象+按照变化顺序写
- 想象练习：火烧云还会变成什么？按照"出现-变化-消失"的顺序说一说
  - 示例：一会儿来了一只小兔子，______，忽然______，一转眼______。
- 学生在手册上画一画、写一写自己想到的火烧云形状

##### 4. 小结收尾（5分钟）
- 火烧云下去了，作者说"一会儿工夫，火烧云就下去了"，给人留下无限遐想
- 指导书写剩余生字

**呼应设计**：延续"时间表"梳理方法，强化"按顺序写变化"这一语文要素，读说写结合。

### 第三阶段：交流探究——寻找奇妙
**共计：2课时**

---

#### 第7课时：交流平台+初试身手
**学习目标**：
1. 交流单元学习收获，梳理"按一定顺序写"的方法
2. 初试身手，练习观察描写身边的奇妙

**教学过程**：

##### 1. 交流平台（15分钟）
- 四人小组交流：
  - 这几篇课文都写了什么？
  - 作者们是按照什么顺序写的？
  - 你学到了哪些描写景物的方法？
- 全班梳理总结：

  | 课文 | 写作顺序 |
  |------|----------|
  | 我们奇妙的世界 | 从天空到大地，按时间/季节顺序 |
  | 暴风雨来临之前 | 按时间推移顺序（前-中-后） |
  | 火烧云 | 按时间变化顺序（上来-变化-下去） |

- 小结：**按照一定顺序写，文章条理清楚，读者一目了然**

##### 2. 初试身手："我的奇妙发现"（20分钟）
- 课本练习：你在生活中发现过哪些"看得见的音乐"？
- 现场观察：观察教室，说说哪里奇妙？按照什么顺序说？
- 练习写一段话：选择你身边一个奇妙的景物，按照一定顺序写下来
- 同桌互相评一评：是不是按顺序写的？

##### 3. 布置观察任务（5分钟）
- 明天我们进行"奇妙世界户外寻找"活动
- 每人留心观察校园或小区，找到一处你认为奇妙的景物
- 按照一定顺序记在观察手册上

---

#### 第8课时：综合性学习——"寻找身边的奇妙"
**学习目标**：
1. 运用学到的观察方法，在生活中寻找奇妙
2. 练习按照一定顺序记录自己的发现
3. 交流分享，互相启发

**教学过程**：

##### 1. 明确任务（5分钟）
- 回顾：我们学到了什么方法？（仔细观察、按顺序写）
- 任务：走出教室，寻找身边的奇妙

##### 2. 实地观察（20分钟）
- 学生分组到校园寻找奇妙
- 要求：
  1. 找到一处你认为奇妙的景物
  2. 看看它是怎样变化的
  3. 按照顺序记录在《奇妙世界观察手册》上
  4. 可以画简单示意图

##### 3. 分享交流（15分钟）
- 小组分享：我找到了什么奇妙？我是按什么顺序观察的？
- 互评：他说清楚了吗？顺序对吗？
- 挑选几份优秀观察记录全班展示

**呼应设计**：体现"语文-生活"呼应，从课本走向生活，在实践中运用学到的方法。

### 第四阶段：表达展示——记录奇妙
**共计：2课时**

---

#### 第9-10课时：习作《我的奇思妙想》
**学习目标**：
1. 激发学生奇思妙想，培养想象力
2. 能按照一定顺序介绍自己的奇思妙想
3. 完成习作，修改自己的习作

**教学过程**：

##### 第1课时：审题构思+起草
###### 1. 激趣导入（5分钟）
- 谈话：这个单元我们发现了大自然的奇妙，其实我们也可以有自己的奇思妙想，创造奇妙
- 审题：什么是"奇思妙想"？（奇妙的想法，不一样的想法）

###### 2. 启发思路（15分钟）
- 课本例子启发：
  - 会飞的木屋
  - 水上行走鞋
  - 会变大变小的书包
- 头脑风暴：你有什么奇思妙想？
  - 想发明什么？
  - 它是什么样子的？
  - 有什么奇特功能？
- 小组交流：说说自己的奇思妙想

###### 3. 构思提纲（10分钟）
- 写作顺序建议：
  1. **为什么会有这个想法**（开头）
  2. **它是什么样子的**（按一定顺序介绍外形）
  3. **它有什么功能**（分点介绍功能）
  4. **这个奇思妙想给我带来什么**（结尾）
- 学生在手册上填写写作提纲

###### 4. 学生起草（15分钟）
- 学生开始写作文，教师巡视指导

##### 第2课时：修改讲评+誊写
###### 1. 佳作示例欣赏（5分钟）
- 读一两篇优秀习作，说说好在哪里？
- 特别关注：是不是按顺序写的？

###### 2. 自我修改（10分钟）
- 修改要求：
  - 读一读，通顺吗？
  - 想一想，顺序清楚吗？
  - 改一改，错别字和标点
- 同桌互相提修改建议

###### 3. 完成习作（25分钟）
- 学生誊写作文到作文本
- 同时完成《奇妙世界观察手册》最后一页：我的奇思妙想

###### 4. 小结（5分钟）
- 奇妙的世界等着我们去发现，奇妙的未来等着我们去创造

---

#### 第11课时：单元总结+成果展示
**学习目标**：
1. 展示单元学习成果
2. 总结单元学习收获
3. 评价学习情况

**教学过程**：

##### 1. 成果展示："奇妙世界观察手册"博览会（20分钟）
- 每个同学把自己完成的《奇妙世界观察手册》放在桌面上
- 全班同学自由参观、阅读他人的手册
- 每人投出三票：
  - 最奇妙发现
  - 最清晰顺序
  - 最美观手册

##### 2. 单元总结（15分钟）
- 师生共同回顾：
  - 这个单元我们学到了什么？
  - 你最大的收获是什么？
  - 今后你还会怎样寻找奇妙？
- 教师总结：
  > 世界充满了奇妙，只要我们**仔细观察**，**按照一定顺序**描写，就能发现美、表达美。希望大家今后继续在生活中寻找奇妙！

##### 3. 单元评价（5分钟）
- 学生自我评价：这个单元我表现怎么样？
- 教师评价，表扬进步学生

## 四、单元评价设计

### 过程性评价（40%）
| 评价项目 | 评价要点 | 分值 |
|----------|----------|------|

| 奇妙手册 | 完成认真，记录清楚 | 15 |

- **顺序清楚**（20分）：是否按照一定顺序写
- **内容具体**（15分）：有没有把奇思妙想说清楚
- **语句通顺**（5分）：语句通顺，错别字少

### 综合性学习评价（20%）
- 观察参与度（10分）
- 交流表达（10分）

## 五、设计说明（呼应《呼应学习任务群》理念）

### 1. 呼应"任务群"理念
- 本设计以**"制作《奇妙世界观察手册》"**这一核心任务驱动整个单元学习
- 贯穿"语文-生活-语文"呼应：从语文学习方法→生活中实践运用→回到语文表达成果
- 体现"实用性阅读与交流"任务群要求

### 2. 呼应"大单元整体设计"
- 单元整体规划，11课时连贯推进
- 语文要素**"按照一定顺序写景物"**一以贯之
- 从"领悟方法→实践运用→表达输出"层层递进

### 3. 呼应"呼应"教学设计风格
- 每篇课文都遵循**"发现-感受-分享-运用"**四环节
- 多用**"表格梳理、时间线"**帮助学生理清顺序
- 重视**语文实践**，学生动手动脑动嘴
- 最后的综合性成果**《奇妙世界观察手册》**，呼应单元开篇任务，前后呼应

### 4. 体现新课程标准要求
- 识字写字为基础，重视阅读
- 关注学生观察能力和想象能力培养
- 重视实践性，语文学习联系生活
- 循序渐进，符合三年级学生认知水平
