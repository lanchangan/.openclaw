### 2. FinLab ⭐⭐⭐⭐⭐

**来源：** ClawHub  
**链接：** https://clawhub.ai/koreal6803/finlab

**功能特点：**

| 功能模块 | 说明 |
|---------|------|
| 📊 量化框架 | 台湾股票市场专用专业框架 |
| 🧪 回测引擎 | 支持完整策略回测 |
| 📉 风险控制 | 止损、止盈、移动止损 |
| 🔧 技术指标 | RSI、MACD、MA等内置指标 |
| 🤖 自动交易 | 支持券商对接下单 |

**数据获取：**
```python
from finlab import data

# 价格数据
close = data.get("price:收盤價")
volume = data.get("price:成交股數")

# 财务数据
roe = data.get("fundamental_features:ROE稅後")
revenue = data.get("monthly_revenue:當月營收")

# 估值数据
pe = data.get("price_earning_ratio:本益比")
pb = data.get("price_earning_ratio:股價淨值比")
```

**策略回测示例：**
```python
from finlab import data
from finlab.backtest import sim

# 获取数据
close = data.get("price:收盤價")
pb = data.get("price_earning_ratio:股價淨值比")

# 构建策略
cond1 = close.rise(10)  # 10日上涨
cond2 = close > close.average(60)  # 站上60日均线
cond3 = pb.rank(axis=1, pct=True) < 0.3  # 低估值

position = cond1 & cond2 & cond3
position = pb[position].is_smallest(10)  # 选最低估值10只

# 回测
report = sim(position, resample="M", upload=False)
print(f"年化收益: {report.metrics.annual_return():.2%}")
print(f"夏普比率: {report.metrics.sharpe_ratio():.2f}")
print(f"最大回撤: {report.metrics.max_drawdown():.2%}")
```

**安装方式：**
```bash

```

### 3. Backtest Expert ⭐⭐⭐⭐

**来源：** ClawHub  
**链接：** https://clawhub.ai/veeramanikandanr48/backtest-expert

**功能特点：**

| 功能模块 | 说明 |
|---------|------|
| 🎯 专业方法论 | 避免过拟合的系统化回测流程 |
| 🔍 压力测试 | 参数敏感性分析、滑点模拟 |
| ⚠️ 风险识别 | 常见陷阱检测（前视偏差、生存偏差） |
| 📚 最佳实践 | 策略验证标准化流程 |

**核心理念：**
> "找到'最不容易失败'的策略，而不是'纸上利润最高'的策略"

**回测工作流：**
1. **陈述假设** - 用一句话定义你的交易优势
2. **编码规则** - 零主观判断，完全规则化
3. **初始回测** - 最少5年数据，覆盖牛熊周期
4. **压力测试** - 80%时间花在这里
   - 参数敏感性：测试 ±50% 参数范围
   - 执行摩擦：1.5-2倍滑点
   - 时间稳健性：逐年分析
5. **样本外验证** - 滚动窗口测试

### 4. Ptrade ⭐⭐⭐⭐

**来源：** ClawHub  
**链接：** https://clawhub.ai/coderwpf/ptrade

**功能特点：**

| 功能模块 | 说明 |
|---------|------|
| 🔄 自动下单 | 支持A股实盘交易 |
| 📊 交易函数 | order(), order_value(), order_market() |
| 💰 融资融券 | margin_trade(), margincash_open() |
| ⏰ 盘后交易 | after_trading_order() |

**交易函数示例：**
```python
# 限价买入
order('600570.SS', 100, limit_price=39.0)

# 按金额买入
order_value('600570.SS', 100000)  # 买入10万元

# 市价单（上交所需要限价保护）
order_market('600570.SS', 100, 0, limit_price=35.0)

# 融资买入
margincash_open('600570.SS', 1000, limit_price=39.0)

# 盘后固定价格
after_trading_order('600570.SS', 100)
```

## ⚠️ 重要提示

### 关于实盘交易的风险
1. **不要让 AI 直接下单** - OpenClaw 作为信号生成器，人工决策
2. **物理隔离** - 交易主机不放 OpenClaw，通过消息队列连接
3. **回测不能代表未来** - 即使通过压力测试的策略也可能失败
4. **小资金验证** - 先用小资金测试，再逐步放大

### 常见陷阱
- ❌ 过拟合：策略只在历史数据表现好
- ❌ 前视偏差：使用了当时不可知的信息
- ❌ 生存偏差：只测试现存股票，忽略已退市
- ❌ 滑点低估：实际成交比回测差很多
