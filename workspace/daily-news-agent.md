# 每日新闻与市场早报代理配置

## 任务描述
每天早上7:30执行以下任务：

### 1. 伊朗-美国战争最新消息
- 使用 web_search 或 browser 搜索
- 重点关注：全球经济影响、霍尔木兹海峡局势、油价波动

### 2. 半导体行业新闻
- 搜索关键词：半导体、WiFi芯片、芯片验证、ESP32、乐鑫
- 重点关注行业动态和技术趋势

### 3. 乐鑫科技(688018)股票分析
- 获取最新股价、涨跌幅
- 搜索公司最新消息和公告
- 结合国际形势（如伊朗战争对半导体供应链影响）
- 给出持有/卖出的建议

### 4. 国内新闻TOP10
- 搜索国内最新热点新闻
- 筛选最重要的10条

### 5. 国际新闻TOP10
- 搜索国际最新热点新闻
- 筛选最重要的10条

### 6. 股票推荐TOP10
- 使用 available_skills 中可用的股票分析技能
- 推荐10个值得买入的股票及理由

### 7. 基金推荐TOP10
- 使用 available_skills 中可用的基金分析技能
- 推荐10个值得买入的基金及理由

## 输出格式
将所有信息整理为卡片形式，通过飞书发送给主人。

## 执行方式
- 作为子代理(session_spawn)运行
- 每天早上7:30触发
- 完成后通过 message 发送结果到主人的飞书
