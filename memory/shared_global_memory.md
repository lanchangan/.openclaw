## 🚨 严重错误警示 - 必读

### ⚠️ 飞书文档操作三大错误（2026-03-31）

> **这是三次严重错误的记录，所有 agent 必须牢记！**

---

#### ❌ 错误一：文档内容为空

**现象：** 创建了飞书文档，但只有标题，没有内容

**根因：** 只调用了 `feishu_doc create`，**忘记调用 `feishu_doc write`**

**必须这样做：**
```json
// 步骤1: 创建文档
{ "action": "create", "title": "文档标题", "owner_open_id": "ou_xxx" }
// 返回 doc_token: "ABC123"

// 步骤2: 写入内容 ⚠️ 这一步绝对不能忘！
{ "action": "write", "doc_token": "ABC123", "content": "# 标题\n\n完整内容..." }

// 步骤3: 获取链接并发送
// 链接格式: https://feishu.cn/docx/{doc_token}
```

---

#### ❌ 错误二：缺少新闻来源链接

**现象：** 文档有内容，但每条新闻没有来源链接

**根因：** 从搜索结果摘要复制内容时，**没有保留完整的 URL**

**必须这样做：**
```
正确格式：
📰 [新闻标题]
[新闻摘要内容]
🔗 来源：[网站名称] | [完整URL]

示例：
📰 中东战事致燃油短缺越南多家航司将削减航班数量
因中东战事引发燃油供应吃紧、价格飙升，越南多家航空公司宣布将于4月起削减航班...
🔗 来源：腾讯新闻 | https://view.inews.qq.com/a/20260329A02RIQ00
```

---

#### ❌ 错误三：使用百度跳转链接而非真正的源链接

**现象：** 新闻链接显示为 `http://www.baidu.com/link...`，无法访问

**根因：** 百度搜索结果中的链接是**跳转链接**，不是真正的源 URL

**为什么这是问题：**
- 百度链接格式：`http://www.baidu.com/link?url=xxx`
- 这是百度的跳转服务器，用户点击后才能跳转到真实网站
- 直接访问这个链接可能失败或显示错误

**必须这样做：**
```
❌ 错误链接：
🔗 来源：半导体行业观察 | http://www.baidu.com/link?url=xxx

✅ 正确链接：
🔗 来源：Reuters（路透社）| https://www.reuters.com/world/asia-pacific/chinas-no-2-chipmaker-readies-7-nm-production-2026-03-16/
```

**解决方案：**
1. **优先使用 web_search (Tavily)** - 返回的链接是真实 URL
2. **如果使用百度搜索** - 需要用 web_fetch 访问跳转链接获取真实 URL
3. **验证链接** - 确保链接是可直接访问的完整域名

### 2. 飞书文档操作最佳实践

**创建并写入文档的完整流程：**

1. **创建文档**
   ```json
   {
     "action": "create",
     "title": "文档标题",
     "owner_open_id": "ou_xxx"
   }
   ```

2. **写入内容**（必须执行！）
   ```json
   {
     "action": "write",
     "doc_token": "ABC123",
     "content": "# 标题\n\n内容..."
   }
   ```

3. **获取文档链接**
   - 格式：`https://<domain>.feishu.cn/docx/<doc_token>`

4. **发送消息**
   - 将文档链接附在消息中

**注意事项：**
- `create` 只创建空文档，必须调用 `write` 写入内容
- `owner_open_id` 确保用户有编辑权限
