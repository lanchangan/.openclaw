# Skill Vetting - 技能安全审核工具

## 核心功能

### 🔍 安全扫描
- 扫描技能中的所有代码文件，识别恶意代码模式
- 检查是否有可疑的网络请求、文件操作、系统命令执行
- 检测是否存在隐私数据泄露风险
- 识别权限滥用和越权操作

### 📊 风险评估
- 对每个技能进行安全评级（安全/低风险/中风险/高风险）
- 生成详细的安全评估报告
- 列出发现的所有安全问题和修复建议
- 计算风险得分，决定是否允许安装

### 🛡️ 自动防护
- 自动拦截高风险技能的安装
- 对中风险技能进行沙箱隔离运行
- 监控已安装技能的运行行为
- 发现异常行为自动告警并阻止

### 📝 审核记录
- 记录所有技能的审核历史
- 保存安全评估报告
- 跟踪技能版本更新的安全变化
- 生成安全审计日志

## 使用场景

**自动触发场景：**
1. 安装新技能前自动进行安全审核
2. 技能更新时自动重新审核
3. 定期扫描已安装技能的安全状态
4. 发现可疑行为时触发紧急审核

## 执行流程

### 1. 静态代码扫描
- 分析SKILL.md文件，检查技能描述和权限声明
- 扫描所有脚本文件，查找恶意代码特征
- 检查配置文件中的可疑设置
- 识别硬编码的敏感信息（密钥、密码等）

### 2. 动态行为分析
- 在沙箱中运行技能，监控所有系统调用
- 记录网络请求、文件操作、命令执行等行为
- 对比声明的权限和实际行为是否一致
- 检测是否有隐藏的恶意行为

### 3. 风险评估
- 根据扫描结果计算风险得分
- 生成安全评级：
  - ✅ 安全：无任何风险，可正常安装
  - ⚠️ 低风险：存在 minor 问题，不影响正常使用
  - 🟡 中风险：需要用户确认后才能安装
  - 🔴 高风险：存在严重安全问题，自动阻止安装

### 4. 报告生成
- 生成详细的安全评估报告
- 列出所有发现的安全问题和具体位置
- 提供修复建议和解决方案
- 保存审核记录到审计日志

## 安全规则

### 禁止的行为
- 未经授权的网络请求到外部服务器
- 读取或修改系统敏感文件
- 执行未声明的系统命令
- 收集或上传用户隐私数据
- 隐藏真实功能的恶意代码
- 越权访问其他应用的数据

### 允许的行为
- 访问声明的API接口
- 操作指定目录下的文件
- 执行声明的系统命令
- 收集必要的功能数据
- 正常的技能功能实现

## 配置选项

```yaml
# 审核严格程度：low/medium/high
strictness: medium
# 是否自动阻止高风险技能
auto_block_high_risk: true
# 是否自动隔离中风险技能
auto_isolate_medium_risk: false
# 扫描超时时间（秒）
scan_timeout: 30
# 保留的审核记录数量
max_audit_records: 100
```
