# Summarize 智能摘要技能

## 核心功能
- 📄 **文本摘要**：支持任意长度文本的摘要生成，保留核心信息
- 📑 **文档摘要**：支持PDF、Word、Markdown等多种文档格式的摘要
- 🌐 **网页摘要**：自动提取网页内容并生成摘要
- 💬 **聊天摘要**：总结长对话、会议记录、聊天历史的核心内容
- 🎯 **要点提取**：自动提取内容的核心要点和关键信息
- 🔑 **关键词提取**：提炼内容的核心关键词和主题
- ⚡ **自定义长度**：支持短摘要（100字）、中摘要（300字）、长摘要（1000字）等不同长度

## 摘要算法
- **抽取式摘要**：从原文中提取最重要的句子组成摘要
- **生成式摘要**：使用大模型重新组织语言生成流畅的摘要
- **混合模式**：结合两种模式的优点，既准确又流畅
