# 自我优化技能

## 核心功能

### 1. 记忆优化
- 定期梳理daily memory文件，将重要信息提炼到MEMORY.md
- 删除过期和无关的记忆内容，保持记忆库精简高效
- 自动分类和标签化记忆内容，提升检索效率

### 2. 流程优化
- 记录每次任务的执行时间和效果，分析瓶颈
- 自动优化多步骤任务的执行顺序，减少冗余操作
- 总结常用操作模式，形成可复用的工作流模板

### 3. 技能学习
- 自动发现缺失的技能，主动从clawhub安装
- 学习新工具的使用方法，更新到本地TOOLS.md
- 记录工具使用经验和最佳实践，提升使用效率

### 4. 自我诊断
- 定期检查系统状态和工具可用性
- 自动修复常见配置问题和权限问题
- 记录错误日志并分析根本原因，避免重复犯错

## 使用场景

**当以下情况时自动触发：**
1. 空闲时间（无用户任务时）
2. 任务执行完成后（进行复盘）
3. 遇到错误或失败后（进行诊断修复）
4. 定期（每天/每周）进行整体优化

## 执行流程

### 记忆优化流程
1. 读取最近7天的memory/YYYY-MM-DD.md文件
2. 提取重要事件、决策、经验教训
3. 更新MEMORY.md，删除重复和过期内容
4. 压缩旧的记忆文件，归档到archive/目录

### 流程优化流程
1. 分析最近10次任务的执行记录
2. 识别重复操作和低效步骤
3. 生成优化后的工作流模板
4. 更新SKILL.md和TOOLS.md中的最佳实践

### 技能学习流程
1. 检查最近任务中用到的工具和技能
2. 搜索clawhub是否有相关的优化技能
3. 安装并学习新技能，更新本地技能库
4. 测试新技能的可用性和效果

### 自我诊断流程
1. 检查所有工具的配置和权限
2. 运行诊断命令验证工具可用性
3. 修复发现的问题，更新配置文件
4. 生成诊断报告，记录修复过程
